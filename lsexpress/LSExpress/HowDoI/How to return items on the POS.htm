How to return items in the POS

You can do returning items in several ways; by setting the quantity to negative, returning an item from the sales history, or reversing an entire sale from the sales history.

Negative quantity

To set the quantity to negative, add the item to the sale.

 

  1. Select the line and open the sales line menu. Change the quantity to a negative number, such as -1, and go back to the POS.

    The item has changed to a returned item, the line colour has changed to red, and the amounts are now negative.

     

  2. Select the Pay action and finalize the returned sale by selecting a payment method for the refund or issue a store credit.

Exchanging an item for a different item

If the customer wants to exchange an item for a different item, add the returned item to the sale and then add the item they wish to exchange it.

 

Then select the Pay action and finalize the sale:

  • If the returned item's price is higher than the item sold, the sale is finalized by selecting a payment method for the refund or a store credit is issued.

  • If the item's price is higher than the returned item, the sale is finalized like a regular sale, and the returned item's price is deducted from the sale.

  • If, however, the total is zero - that is, the customer has exchanged the item for an item of the same value, finalizes the sale by selecting Pay and either select Cash and register a zero cash payment, or select the action menu and Post.

    For more information on how to issue a store credit, see here.

Returning an item from Sales History

  1. Items can also be returned by using the Return Items action.

  2. Once the action is selected, a page will open where the sale items can be selected.

  3. Once all the items that are being returned have been selected, open the action menu and select Proceed.

  4. The items are then added to the POS as returns and can be finalized as usual.

  5. A credit memo will be created and added to the Sales History.

Reversing a sale from Sales History

You can use the reverse sale action to return items and reversing the original payment.

 

  • From the Sales History page (either from the role center or within the POS), select the sale and then open the details menu and select Reverse Sale. The sale posts directly and offers to print or email a receipt.

  • A credit memo will be created and added to the Sales History.
