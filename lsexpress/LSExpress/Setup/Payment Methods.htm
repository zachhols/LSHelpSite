Payment Methods

During setup, the Cash payment method was automatically created, but LS Express supports a various payment methods.

How to add a new payment method

  1. Open the Payment Methods page and select to create a new payment method.

  2. Select a Payment Method Code or create a new code by selecting New in the list.

  3. Select whether Overtender should be allowed or if it should be limited. If Limited is selected, then the Maximum Overtender amount needs to be entered.

  4. The Change Tender Code specifies the payment method to be used for change when overtender is allowed or limited. If nothing is selected, then the same payment method is used for the change amount.

  5. It is also possible to specify whether the Cash Drawer shall open at the end of the sale. Please note that this setting is only available when printing with PrintNode, and the drawer has to be connected to the printer.

  6. If you have added the LS Pay module to your license, you can activate LS Pay for a payment method. By activating the LS Pay toggle, the LS Pay app is woken automatically when the payment method is chosen at the POS by activating the LS Payapp. For more information on setting up LS Pay, see here.

  7. If the payment method is a Store Credit method, a check mark is placed in the Store Credit field. For more information, see below.

  8. The Count at Close Outfield should be checked if the payment method should be counted during the close out process.

  9. It is also possible to add a picture to the payment method. For more information, see below.

How to set up the store credit payment method

The Store Credit payment method is used when issuing store credit for returned goods or when a customer returns with a store credit receipt and wants to use it as payment.

  • To set it up, check the Store Credit field.

  • For more information on how to use store credit, see here.

How to add a picture to a payment method

  1. Select the Payment Method Picture and select to either import a new picture from the file or take a picture if the device allows it.

  2. Locate the image and upload it, and from now on, when accepting payments on the POS, you can click on the picture.

How to configure the gift card payment method

Note: for a gift card to work as payment, you need to add cash value to the gift card.

  1. Add a new Payment Method for receiving gift cards. Path: Administration > Navigate > Payment Methods.

  2. In the Settings tab, make sure you turn on the Gift Card option.

Steps to sell Giftcard:

  1. Go to Items, click on New and create a new item that will be used for your gift cards.

  2. In the Item, tab changes the Type to non-inventory.

  3. In the LS Express tab, activate the Gift Card option.

    Once you have followed these steps, you should be able to sell and receive gift cards at the POS.
