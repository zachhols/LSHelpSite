Payment Methods

During setup the Cash payment method was automatically created, but LS Express supports a variety of different payment methods.

 

How to add a new payment method

Open the Payment Methods page and select to create a new payment method.

 

Select a payment method code or create a new code by selecting new in the list.

 

 

 

Select whether overtender should be allowed, or if it should be limited. If limited is selected then the maximum overtender amount needs to be entered.

 

The Change Tender code specifies the payment method to be used for change when overtender is allowed or limited. If nothing is selected then the same payment method is used for the change amount.

 

It is also possible to specify whether the cash drawer should be opened at the end of the sale. Please note that this setting is only available when printing with PrintNode and the drawer has to be connected to the printer.

 

If the payment method is a store credit method a check mark is placed in the Store Credit field. For more information see below.

 

The Count at Close Out field should be checked if the payment method should be counted during the close out process.

 

It is also possible to add a picture to the payment method. For more information see below.

 

How to set up the store credit payment method

The Store Credit payment method is used both when issuing store credit for returned goods, or when a customer returns with a store credit receipt and wants to use it as payment.

 

To set it up simply check the store credit field.

 

 

For more information on how to use store credit see here.

 

 

How to add a picture to a payment method

Select the Payment Method Picture and select to either import a new picture from file or - if the device allows it - take a picture.

 

 

Simply locate the image and upload it, and from now on when accepting payments on the POS you can simply click on the picture.

 

 

How to configure the gift card payment method

Note: for a gift card to work as payment, you need to add cash value to the gift card.

 

  • First, add a new Payment Method for receiving giftcards. Path: Administration > Navigate > Payment Methods
  • In the Settings tab make sure you turn on the Gift Card option.

 

 

Return to your store´s home screen

 

In order to sell gift cards there are a few steps you need to do:

  • Next go to Items, click on New and create a new item for your gift card.
  • In Item tab change the Type to non-inventory.
  • Activate the Gift Card option under the LS Express tab.
