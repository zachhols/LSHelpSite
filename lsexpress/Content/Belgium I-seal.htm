Belgium i-Seal

Localization setup

  1. In LS Express, search for “BE Localization Setup”

  2. Turn on the Print Seal No. Boolean to allow the Seal number to be printed on the receipt.

  3. The Prefix of Seal File can be defined with any name.

  4. Define the Journal Template Name and the Journal Batch Name fields in the General Journal fast tab.

Disable Seal functionality

In a Subscription Setup,

Path: Administration > Related > Subscription.

 

From the Point of Sales List, the Seal functionality will disable if you tick the Disable Seal Functionality button. Hence, the i-Seal transaction record will not be generated whenever a sale is posted.

 

If you leave the field blank, it will automatically generate the i-Seal transaction record.

Sales History

The sales history enables you to keep track of items history when you make normal sales, reverse sales, or return items. In the right column of this table, you can track the Seal Number of each transaction. When you click the Seal number, it will navigate to the i-Seal transactions page.

POS receipt printing

By enabling the Print Seal No., the receipt slip and invoice will print the Seal number once the transaction is posted.

In case you turn off the Print Seal No. Boolean in the Localization setup, the receipt slip and invoice will not print the Seal number.

However, Seal No. will still generate in the i-Seal transaction entries.

 

For more information on how to modify the layout of the POS receipt in word, click Receipt Layout.

i-Seal transactions

  1. Open the i-Seal Transaction List.

  2. Once a transaction has been posted in the POS, the table will display the transaction entries with Seal Number. The first transaction will generate zero Seal numbers, and the following transaction will generate a unique number. This Seal number is generated either by performing a normal, refund or credit memo transaction. The Seal Number will not generate if no transaction is posted.

Export i-Seal transaction

  1. The highlighted rows show the transactions are not exported yet. To export the transactions, select Export.

  2. Once a transaction has been exported, LS Express will upload the file information to the csv file.

  3. The Date Time exported, Exported by, and Seal exported information is now updated.

i-Seal transaction export report

  1. Search Transaction Export Report.

  2. Select OK to export all the transactions.

  3. The export report is now generated in the text file. One receipt can have multiple lines.
