LS Pay setup

LS Pay is an EFT (Electronic Fund Transfer) software solution that lets you accept card payments. For general information about LS Pay, see here.


Using LS Pay with LS Express involves three key components:

  • Subscription. LS Pay must be included in your LS Express subscription. This subscription lets you configure which payment methods require LS Pay authorization.
  • LS Pay App. Once you accept a payment with a payment method that has been configured to require LS Pay authorization, the authorization request is sent to the LS Pay app.
  • EFT terminal. The LS Pay app communicates with the EFT terminal that captures the card transaction.

Have a look at the sections below on how to configure each component.

Subscription

Your subscription must include LS Pay. For more information, please contact your LS Express supplier.
You can try out LS Pay in a Cronus demo company without adding it to your subscription. See LS Pay testing without a subscription for more information.

LS Pay app

The LS Pay app is available to download for Windows. To get access, please submit a support request through the LS Retail partner portal.

EFT terminal

LS Pay gives you the flexibility to choose from a variety of Payment Service Providers (PSP), with out-of-the-box support with PSP's in several regions. For information on supported PSP's by region please log into the LS Retail partner portal. For general information on LS Pay, see here.

LS Pay testing without a subscription

You can test LS Pay in a Cronus demo company without a subscription: 

  • From the LS Express role center, open the Administration page and select to Edit the page.

  • The LS Pay Testing action on the Actions menu enables LS Pay in a testing environment.

Note: This action is only visible in a Cronus demo company.

  • When you invoke the action, the current setting will display with an option to change it.

  • When LS Pay Testing is enabled, the option to test against the LS Pay app is offered.

  • If you choose not to test against the app, you can only test the LS Pay flow within LS Express.
  • If you choose to test against the LS Pay app, the LS Pay app is invoked whenever a payment requires authorization, even if the app is not installed on the device in use.

How to set up the LS Pay app

Once you have the LS Pay app installed on the device you wish to use, you are ready to pair the app with the EFT terminal: 

 

  1. Open the app, and a login screen appears. At this time, the password is an unchangeable 1234.

  1. Once you are logged in, the main screen is displayed.

Note: At this point, make sure that the device you are using and the EFT terminal you wish to pair are on the same network.

 

To configure a VerifonePoint device type: 

 

  1. Make sure you select the correct device type in LS Pay in the Choose device type field.
  2. Tap Device Settings to configure the app for your EFT terminal.

  1. Enter all the settings:
    • When you turn on the terminal, the IP address of the terminal is displayed.
    • Main Port: Default port is 5015.
    • IP:Secondary Port: The default port is 5016.
    • Timeout Value: This is the timeout between the app and the EFT terminal. Default is 10 seconds.

  1. Go back to the main screen, and tap Pair Device. A dialog box with a code for the EFT terminal is displayed.

  1. Enter the code into the EFT terminal to complete a successful pairing.

How to activate LS Pay for a payment method

For LS Pay to be triggered in the POS, you need to configure a payment method to trigger the LS Pay app.

  • You can either add LS Pay to an existing payment method or create a new one.

  • Once LS Pay has been added to the subscription (or LS Pay Testing has been enabled for Cronus), a check box is visible in the Payment Method's Settings.

Note: It is possible to have multiple payment methods that use LS Pay

  • When you choose the payment method in the POS, one with LS Pay Payment activated, you should get the option to Open LS Pay for LS Express.

    The LS Pay should open and process the payment.

Accepting payment

Once LS Pay has been set up, it can be used with LS Express. See How to use LS Pay for more information.

Troubleshooting

A common pitfall when dealing with LS Pay in LS Express:

When I accept an LS Pay payment, nothing happens, or a new empty tab is opened.

This typically happens when the app is not installed on the device being used.
