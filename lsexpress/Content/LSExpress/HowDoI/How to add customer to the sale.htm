Customers

Note that if you buy any item and post with exact cash, the system will post the invoice and the payment. It will use the payment to knock off the invoice together. Whether it is cash, credit card or gift card, when you buy something and post the payment together, it completes the wholesale.

 

LS Express is set up to use a cash customer by default. It is possible to change the default customer. See Default Customer for more information.

How to add a different customer to the sale

  • To add a customer to the sale, open either My Customers or All Customers from the POS. Then select the customer.

  • Once the sale is finalized, the Posted Sales Invoice is linked to the customer's account.

Walk In Customer

In POS, when the Customer name is set to Walk In Customer, and they buy items, to post only invoice is not allowed. In the actions menu, there is no option for Post to Customer. This is due to the system already set that Walk In Customer is the default customer, which means they pay cash.

How to determine it is cash

Log in as admin, and every user enable to 1 POS device license.

 

Path: Administration > Navigate > Subscription

 

  1. Every POS must attach to 1 user ID. Now, you log in as admin. So, this admin will link with the Sales Profile. For example, there are 3 POS in Sales Profile List, so there will be 3 users ID.

    Note that the Sales Profile is attached to 1 store. For example, you can also have 3 Sales Profile with the same store number.

     

  2. From Sales Profile Card, you can control Printing and pointing to which Store number.

  3. Go to store "ST01", and from the Point of Sale, see POS Customer Profile. This is a Walk In Customer. After posted or made a payment, the system will automatically create another blank POS sales with this default customer and default location code.

    This is the reason why Post to Customer's Account created.

How to view the customer's sales history from the POS

Path: POS > Actions > LS Express > Customer Item Sales History.

 

Here you will see the items that the customer has purchased in the past and an option to open the posted document or print/email the receipt. The sales are only showing the history of customers with the cash payment.

How to block a sale if a customer's credit limit has exceeded

You can specify if a customer is not allowed to exceed his/her credit limit at the POS.

 

  • Open the Customer Card. In the Point of Sale tab, turn on Block Customer Credit Limit at POS.

  • If you try to post the sale to the customer, it will be blocked.

    Note: If the sale with payment together, we still allowed to post sale transaction.
