What's New

LS Express is constantly evolving. Here you can see highlights of the most extensive changes and improvements in each version.

Version 6

V6.0.7

Fast search engine of items name and list in POS Item Input.

V6.0.6

Added LSESearch Items - LSESearch in the Items list.

Added Order Reference No in POS sales page.

 

V6.0.5

Added "Block Return and Negative Quantity" in Staff page.

Now you can do a Refund to customer account for an invoice using the LS Pay.

V6.0.4

Added “Total Payments to Cust. Acc”, Total Sales Amount Incl. Tax & Sales amount functionality in Close Out Summaries page.

V6.0.3

New UI for Item Input in Type the item description, item number or barcode

Added Work Date functionality in Sales Posting Method.

New feature! Now you can Refund into a Customer's Account.

V6.0.1

Added Store Record Level functionality in Security Filters.

Version 5

V5.1.0

Added Reason Code in Sales credit memo before posting.

New Feature! German Fiscalization is available. You can follow How to configure the Fiskaly connector for LS Express in Germany

V5.0.1

Added External Document No. in Payment into account page.

Version 4

V4.1.0

Added functionality for Posting a partial payment into account.

V4.0.0

You can now connect your LS Express store to a Shopify website.

Version 3

V3.1.0

New feature! Now you can create and configure How to create a Promotion.

V3.0.0

Added functionality for partners to upload their Custom Reports: Sales Invoice Slip and Sales Credit Memo Slip.

Version 2

V2.3.2

  • Print a slip when adding to a drawer or removing from a drawer, using PDF or PrintNode.

  • Block a customer if the credit limit has exceeded.

V2.3.1

  • Print a merchant's copy of sales slips (only if LS Pay has been enabled).

  • Stockout Warning at POS.

V2.3.0

  • Use the Quick Cash shortcut to make the cash payment faster.
  • Prevent Negative Inventory.
  • Warning displayed if closing out from another store.
  • Add to & remove from drawer supporting PDF printing.*

V2.2.0

  • Show Sales Lines in Item List

V2.1.0

  • Gift card as a payment method
  • Sell gift cards to customers
  • Use a Gift Card as payment.

V2.0.0

  • Use the Kitchen Printing module to print the order list in a different department.
  • Use the Filtered Item List per Sales Profile to separate the item list for different business.
  • Item Variant is supported in POS.
  • Item Tracking is supported in POS.
  • Returned Item Log to keep track of the history of transactions.
