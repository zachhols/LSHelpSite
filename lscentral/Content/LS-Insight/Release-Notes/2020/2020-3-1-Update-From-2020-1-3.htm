Update LS Insight from v. 2020.1-2020.3 to v. 2020.3.1

In the 2020.3.1 hotfix of LS Insight a pipeline was added that has been missing since LS Insight was updated from ADF version 1 to version 2.

To add this pipeline to a current installation of LS Insight, make sure that you are running LS Insight version 2020.1 or later. Then follow the steps below.

Download update

You can download the update package from our Azure storage.

The zip file contains one file and one folder:

  • ApplyFixes.ps1

    This file is the script you need to run to apply the pipeline updates to your current installation.

  • AllPipelineFixes

    This folder contains the .json files containing the updated pipelines.

Update LS Insight

To implement the fix:

  1. Run the ApplyFixes.ps1 script in VS Code.
  2. The script will prompt for the full paths to three folders:
    • Your original LS Insight base folder containing the parameters.txt file
    • The project folder created when LS Insight was originally set up.
    • The AllPipelineFixes folder
  3. Review the folder names, and answer 'y' if they are correct. If you answer something other, the script will start again from the beginning.
  4. The script will prompt for login to Azure.
  5. The script will run and display a message when finished.

You do not need new Report templates for this update. The only change is in the pipelines, so the data will be updated during the next scheduled pipeline run.
