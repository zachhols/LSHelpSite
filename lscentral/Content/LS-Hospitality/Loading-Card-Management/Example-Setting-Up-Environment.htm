Example: Setting up Environment

The loading card examples take place within store S0008 from the LS demo data. Four new POS terminals were created by duplicating terminal P0018 and the core interface and menu profiles. Each POS was modified to give it a distinctive look.

To set up the POS terminals take a look at Setting up the POS for Loading Card Usage. The Cashier Station has been set up as a cashier terminal. Take a look at Basic Setup to see how that is done.

Displaying the POS Terminal Description

One key change that had to be made was to display the name of the POS terminal on top of the screen. By default different hospitality types are displayed there but when using loading cards all the POS terminals should use the same hospitality type and sales type and therefore it is not necessary to show the hospitality type menu control.

  • Go to the Panel Properties of the main POS panel.

On the panel a new control has been added.

This control contains a menu with the POS terminal description.

This menu has the #TERMINAL_DESCRIPTION tag which contains the POS terminal description.

With this change the header will always contain the description of the terminal being run.

The same thing can be done in the Sales POS to show the description of the terminal instead of the sales type.
