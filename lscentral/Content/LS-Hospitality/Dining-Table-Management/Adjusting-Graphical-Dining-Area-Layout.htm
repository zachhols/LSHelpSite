Adjusting Graphical Dining Area Layout

During opening hours the current layout of the restaurant may have to change when tables need to be moved and/or joined to accommodate the customers.

When you display your dining tables in a graphical layout, you can let the current layout visually represent at any time how the dining tables are placed in the dining area.

The dining area layout you design is stored as the design layout.

The current layout is always the layout that appears in the Hospitality POS client and this layout can be adjusted by staff members that have permission to do so.

In Adjust mode, you perform actions relevant to the current layout, such as moving, rotating, joining dining tables and switching layouts.

Tip: You cannot change the size of dining tables in Adjust mode.

Note: The current layout can be adjusted on only one POS terminal at a time.
