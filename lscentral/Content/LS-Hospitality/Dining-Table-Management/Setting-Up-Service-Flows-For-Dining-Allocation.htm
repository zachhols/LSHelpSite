Setting Up Service Flows for Dining Allocation

To run dining table allocation you must configure some default settings and behavior in case of reservations overlapping. You can also select styles for the dining table list and the waiting list in the Dining Table Allocation panel.

To set up service flow for dining allocation

  1. Go to the Hospitality Service Flow List.
  2. Double-click the service flow that includes dining table allocation.
  3. On the General FastTab, select the Allocation/Reserv. Tracking check box. The system automatically fills in the fields on the Pop-up Menus for Allocation/Reservation FastTab at the bottom of the page.
  4. On the Reservation Status FastTab, you can let the system automatically change the Open status to Late and Late to No-Show after a certain time period.
