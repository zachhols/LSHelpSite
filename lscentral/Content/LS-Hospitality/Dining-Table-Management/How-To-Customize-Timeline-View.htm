How to: Customize the Timeline View

To customize the Timeline view of the Allocation panel, you need to edit the POS menus that are used to build up the timeline button matrix.

The matrix is built with temporary menus but uses the menus specified in the Dining Area Card as a template for how the menus should look and what information should be displayed.

Note: For performance purposes, it is recommended not to have too many rows and columns in the Timeline panel.

Four menus control the look of the Timeline panel: 

  • Time Heading POS Menu
  • Dining Table POS Menu
  • All. Timel. Tbl Status Menu
  • Dining Tbl. Paging Menu

The number of columns in the list are calculated by dividing the View Time Range (Min.) by Reservation Default Time Interval (Min.).

The number of rows (that is, number of dining tables shown) is defined by No. of Dining Tables Shown.

To compare and edit Time Heading and Dining Table heading menus

  1. Go to the Dining Areas page.
  2. Select the dining area in question.
  3. Click Compare Menus - Alloc. Timeline Menus. The Time Heading and Dining Table POS menus appear in the POS Menu Comparison - Header page.
  4. Select these two menus (hold the Ctrl key to multi-select).
  5. Click the Compare Menu Lines for Selected Menus action. In this page you can compare the skin and font settings for the menus and the glyph texts (you can filter on more fields to make the comparison easier) and edit at the same time.

For more information on changing the All. Timel. Tbl Status menu, see How to: Identify Allocation Statuses Visually.

Note: You can set the reservation time interval for each dining period. The Reservation Default time Interval (Min.) is used, if the interval is not set for a dining period.
