KDS Display Stations

A display station shows orders or part of orders on a screen. It can have a bump bar connected that has a set of possible operations.

To set up Display Stations

To check if the Display Stations are correctly configured

See also

A display station can also be a printer station, printing the order or part of the order on a slip.

A display station can act as

  • a preparation station
  • an expeditor station
  • a customer facing station
  • a customer facing display preparation station
  • an upcoming orders station
  • a printer station that prints out a slip with the order or part of the order instead of showing orders on a screen.

Note: It is possible to print the chit on a printer station when the chit is bumped from an Expeditor display station. See Expeditor Bump Printing.

To set up Display Stations

  1. Click the icon, enter Display Stations List, and select the relevant link.
  2. Refer to field tooltips in the Display Station Card page on how to set up a display station.

To check if the Display Stations are correctly configured

  • Click the Process action, and select Validate Setup.

This checks, for example, whether the field Next Display Station ID has been populated for all preparation stations.

LS Display Station Utility helps the user to configure the Display Stations on the computers they are to run on.

Tip: If the computer that runs the Display Stations has multiple screens, you can identify the screens by opening the Display Settings window and clicking the Identify button. This will display the screen number on each screen. Use these numbers when you are deciding the screen number for the display station.

There are two ways to open the Display Settings window: 

  1. Right-click somewhere on the desktop, and select Display settings from the menu.
    or
  2. Open the Settings window from the Windows Start menu, and select Display.

(If the Business Central client is running on a different computer, be sure to be on the computer where the display stations are intended to run.)
