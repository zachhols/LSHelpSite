How to: Design a Chit Display and a Line Display (KDS Display Profile)

The KDS Display Profile defines if the display station is a chit display or a line display. You design lines and columns for each display profile.

Important: If you are configuring a Display Profile for the first time, in an empty system, be sure to insert the KDS default data. See How to: Insert the KDS Default Data

To create a display profile

  1. Click the icon, enter Display Profile, and select the relevant link. Here you define a display mode which is either a Chit Display or a Line Display.
  2. When the profile has been created, you click the KDS Chit Header Designer, KDS Chit Footer Designer, or KDS Line Display Rows Designer action to design lines and columns for that display profile.

In the KDS Chit Header Designer you define lines for the header and columns for those lines. Note the KDS Chit/Line Columns FastTab in the bottom part of the window.

In the KDS Chit Footer Designer you define lines for the footer and columns for those lines. Note the KDS Chit/Line Columns FastTab in the bottom part of the window.

In the KDS Line Display Rows Designer you only define columns for that one line which appears. You cannot add more lines to the line display.

On the KDS Chit/Line Columns FastTab you define what each column in the line stands for. The configurable fields are:

Field Description
KDS Data Type Specifies where to fetch the values from.
  • KOT Data - This is data from the KOT itself, for example KOT no. or Item name.
  • KDS Data - This is system data in the KDS, for example countdown clocks or station letters.
  • Transaction Data - (POS Tags). The POS tags in Business Central can be used directly. See the section on KDS Tags in the Tags topic for more information.
Field Data The data field to be used. The drop-down list differs depending on which KDS Data type is used. The drop-down list also differs for the KOT Data type, depending on whether Chit Display or Line Display is selected.
Field Caption This is a field which is only enabled on line displays.
Width (%) This is the size of each column in the line. The whole line has to equal 100% and you can see the sum of the columns in each line above.
Text Color Choose the text color here. The default value (-1) is the text color from the Order Style is used.
Back Color Choose the background color here. The default value (-1) is the text color from the Order Style is used.
Back Color Transparent Sets the background color to transparent.

Changing the color values will make the field different from the rest of the line.

Note: If no lookup data appears in the Field Data column, the most likely reason is that the lookup data is missing. You can fix this by inserting the default KDS data.
