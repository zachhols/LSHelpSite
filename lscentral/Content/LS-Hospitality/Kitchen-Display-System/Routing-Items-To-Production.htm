Routing Items to Production Sections

KDS routing hierarchy

To set up Item Section Routings

In the system, it is possible to route to production sections for:

  • Recipes, Hospitality Items, Retail Items
  • Special Groups
  • Product Groups
  • Item Categories
  • All Items

It is possible to set up one or more production sections for every recipe, item, group and category.

See also

On KDS Item Routing

KDS routing hierarchy

To find out to which production section the order line’s item is routed, the system looks for the data in this order:

Example:

Recipe A, B and C belong to the same product group. This product group has routing set up for section Cold.

Recipe C belongs to a special group which has routing set up for section Grill.

Recipe A has routing set up for section Fryer.

The system selects the following routing:

  • Recipe A to the Fryer section (recipe routing)
  • Recipe B to the Cold section (product group routing)
  • Recipe C to the Grill section (special group routing)

Note:Production Section routing for all items can only be set up in Item Section Routings.

To set up Item Section Routings

  1. Click the icon, enter Item Section Routings, and select the relevant link.
  2. Click the Edit List action.

Note: In the Code field, you can click the AssistButton to view the card for the item, special group, product group, or item category.

Filtering on Item

You can filter on an item to see which routing lines apply to the item.

You can do this in two ways:

  1. In the Filter on Item field, select the item you want to filter on.
  2. Select a line that is of type Item, and click the Relatedaction, then click Filter on Item.
