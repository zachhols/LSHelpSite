Setting Up Menu Courses

A Coursing KOT contains the complete order for the dining table. The items in the KOT are sorted by menu courses. Each menu course is fired at the appropriate time to notify the kitchen staff that they should start preparing the menu course. The items that belong to the menu course are supposed to be ready at the same time.

Menu Courses are set up within each restaurant.

To create Menu Course menu types

Menu Courses are linked to restaurant menu types. For each menu course, you need to create a restaurant menu type that is used to mark the items in the order. If an item is marked with a menu course menu type, it will be included in the Coursing KOT.

  1. Click the icon, enter Restaurant List, and select the relevant link.
  2. Select the restaurant in question.
  3. Click the Hospitality action, then click Menu Types.
  4. Create a menu type for each course you are going to create, and assign a Code on POS to it. This code appears in the journal lines, displaying the menu type to which an item belongs. The Order of the menu types is important. You can click Up or Down to change the order.

To create Menu Courses

  1. Click the Coursing action for the same restaurant, and then click Rest. Menu Courses.
  2. Create a menu course.
  3. Fill in the Order field. The order is important and the menu courses should appear in the same order as the menu types they are linked to. The actual Order number is not important (menu type 6 can be linked to menu course 1 and so on).
  4. In the Menu Type Order field, select a corresponding restaurant menu type you created in the previous step.
  5. Select the desired Firing Flow for the menu course. It can either be manually fired or fired automatically when the order is announced in the kitchen (on KOT Creation).

To assign Menu Course Tasks

  1. Click the All Courses - Assign All Coursing Tasks action. Each course has now been assigned each menu course task.
  2. If a menu course should not be assigned all the tasks, delete the tasks that do not apply it.
