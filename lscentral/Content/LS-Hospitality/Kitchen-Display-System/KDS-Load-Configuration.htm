KDS Load Configuration

By specifying load configurations for the restaurants in the company, production sections that normally map to several display stations map to the same or fewer display stations when the restaurant is not busy.

A restaurant has 5 display stations:

  • Normal Load - three display stations in use
  • Busy Load - five display stations in use
  • Quiet Load - one display station in use

For the Restaurant you set what is the active load configuration.

  • Production Sections can be mapped to different Display Stations depending on the load configuration.

For the hospitality type you can also set what is the active load configuration.

  • If specified, the system takes the load configuration value from the hospitality type as the active one. If not specified, it takes it from the restaurant.

Create as many Load Configurations as needed for the setup.

To open Load Configuration

  • Click the icon, enter Load Configurations, and select the relevant link.

For each Load Configuration define:

  • Code ID, 20 digits long, alphanumeric field. If Code is not filled in, the load is by default Normal.
  • Description 30 digits long, alphanumeric field.

Tip: There should be one record with a blank Code, indicating the default load configuration. If no active load configuration is selected in the restaurant, the system looks for section display station mappings with a blank load configuration code only.
