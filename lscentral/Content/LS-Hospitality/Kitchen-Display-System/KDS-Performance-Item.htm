KDS Performance Item Report

This report gives an overview of the average production time for items, product groups, or item categories.

The report processes information on posted orders and only includes:

  • KOTs for preparation stations.
  • Items that have a production time (estimated ticket time) greater than 0.
  • Items that have actual ticket time greater than -20 and less than 240 minutes.
  • KOTs that have been bumped on all of its stations.

On the Level and Filtering FastTab, fill in the fields according to these guidelines:

  • Level - select the level you wish to view, item, product group, or item category.
  • Restaurant - if there are multiple restaurants, here you can select which restaurant you wish to view.
  • Today - sets Date From and Date To the today's date.
  • Date From - the date from which you want to run the report.
  • Date To - the date to which you want to run the report. Note: When Date From and Date To are the same date, the report includes entries for all the report periods, also the ones that are past midnight.
  • Report Period - if you are running the report for one day (Date From and Date To are the same), you can limit the report to show results for just one KDS report period.

The KOTs are grouped into report periods by the date and time they are sent to the kitchen.

The report is sorted in alphabetical order by the description of the items, product groups, or item categories.
