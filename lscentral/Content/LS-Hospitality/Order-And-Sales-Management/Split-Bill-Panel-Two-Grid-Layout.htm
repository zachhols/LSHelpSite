Split Bill Panel - Two-Grid Layout

A Split Bill panel with a two-grid layout contains both the table grid and the guest grid.

You can customize the default Split Bill panels regarding the size of the menus, caption, and information bar. You can also add to the fields displayed in the grid forms and change the length of these.

Adjusting sizes, rows, and columns

  1. Go to the POS Panel List.
  2. Filter on Control ID SPLIT to see all the split panels. There should be 6 per profile. The ##DEFAULT profile contains the two-grid layout panels.

On the POS Panel Card, the POS Panel Row/Column FastTab, you can change the values in the size fields. It is best to let the sum of sizes for column and rows add to 100. You need to test your Split Bill panel from the Hospitality POS client for best results.

You can also edit the POS Panels from the POS by right-clicking the Split Bill Panel and selecting Panel Properties. 

The following tables give an overview of the controls in the panels and what can be customized.

Panel controls for two-grid layout

#HO-SPLITBILL-GUEST panel

Control Menu/Panel Info Customize
#DYN-HEAD1 #HO-SPLIT-HDR1-G The header menu with Table and Guest information. Yes
#DYN-HEAD2 #HO-SPLIT-HDR2 Menu with information about the total amount for the table. Yes
#INFO #INFO Information menu for the Panel. No
#SPLITBILL-FUNC #HO-SPLIT-FUNC Main Function menu that contains functions that apply to the panel as a whole. Yes
Panel #HO-SPLITBILL-TABLEG Panel for the table (guest 0). Yes
Panel #HO-SPLITBILL-SPLITG Panel for the guests or seats (guest > 0). Yes

#HO-SPLITBILL-NOGUES panel

Control Menu/Panel Info Customize
#DYN-HEAD1 #HO-SPLIT-HDR1-G The header menu with Table and Guest information. Yes
#DYN-HEAD2 #HO-SPLIT-HDR2 Menu with information about the total amount for the table. Yes
#INFO #INFO Information menu for the Panel. No
#SPLITBILL-FUNC #HO-SPLIT-FUNC Main Function menu that contains functions that apply to the panel as a whole. Yes
Panel #HO-SPLITBILL-TABLE Panel for the table (guest 0). Yes
Panel #HO-SPLITBILL-SPLIT Panel for the guests or seats (guest > 0). Yes

#HO-SPLITBILL-TABLEG and #HO-SPLITBILL-TABLE panels

Control Menu/Data grid Info Customize
#SPLIT-DATAGRIDTABLE #HO-SPLIT-GRID1 The left Data Grid for the Panel. Shows guest = 0 (the table). To add fields to the grid simply add fields to the Data Table. Yes
#DYN-GRID1FUNC #HO-SPLIT-G1FUNC POS Menu between Data Grid 1 and 2 with the split functions for the table. Yes
#DYN-INFO1 #HO-SPLIT-INFO1 POS Menu with total amount not split and marked lines. Yes

#HO-SPLITBILL-SPLITG panel

Control Menu/Data grid Info Customize
#SPLIT-DATAGRIDGUEST #HO-SPLIT-GRID2 The right Data Grid for the Panel. Shows guest > 0. To add fields to the grid simply add fields to the Data Table. Yes
#DYN-GRID2FUNC #HO-SPLIT-G2FUNC Handles functions for Data Grid 2 like “Mark All”, “Print Split Bill”. Yes
#DYN-INFO2 #HO-SPLIT-INFO2 Total amount for the selected guest or seat and marked lines. Yes
#SPLITGUEST-G #HO-SPLIT-GUESTS-G The guest menu that contains 1 button - for guest 1. Yes

#HO-SPLITBILL-SPLIT panel

Control Menu/Data grid Info Customize
#SPLIT-DATAGRID-NOG #HO-SPLIT-GRID2-NG The right Data Grid for the Panel. Shows guest > 0. To add fields to the grid simply add fields to the Data Table. Yes
#DYN-GRID2FUNC #HO-SPLIT-G2FUNC Handles functions for Data Grid 2 like “Mark All”, “Print Split Bill”. Yes
#DYN-INFO2 #HO-SPLIT-INFO2 Total amount for the selected guest/seat and marked lines. Yes
#SPLITNOGUEST-G #HO-SPLIT-NOGUEST-G The guest menu that contains at least 1 button - for guest 1. Yes
