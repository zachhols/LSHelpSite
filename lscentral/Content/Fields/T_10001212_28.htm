Service Type, Hospitality Type

This field indicates the service type of the hospitality type. The service type controls a few important settings for the hospitality type and when you select a service type, some default settings are applied.

The options are:

Not Specified - there is no specific service type, no settings are set

Dining Table Service - Guests sit down at a dining table, orders are taken for the table.

  • Requires a dining area that contains the dining table setup and layout
  • Layout View is either Graphical Dining Tables or Dining Table Grid
  • Order ID is by Dining Table ID
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales
  • On specific POS Terminals, the layout view can be a Dining Table List view. This view is set for the POS terminal, not hospitality type.

Quick Service - Guests order and pay directly. Ordered items may be sent to kitchen for preparation.

  • Layout View is KOT List (kitchen order tickets)
  • Order ID is Queue Counter, Manual Input by Sales Type or Receipt No. Only
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales

Open Orders - Guests order, order stays open and can be added to.

  • Layout View is Order List
  • Order ID is Queue Counter, Manual Input by Sales Type or Receipt No. Only
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales

Takeout Service - Guests call restaurant for takeout or come to restaurant for takeout. Orders are either paid on pick-up or paid on ordering.

  • Layout View is KOT List (kitchen order tickets)
  • Order ID is Queue Counter, Manual Input by Sales Type or Receipt No. Only
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales

Self Service - Guests pick what they want and pay at a POS terminal. This is like normal retail.

  • Layout View is Order List
  • Order ID is Receipt No. Only
  • After Transaction Posted is Stay in Sales POS
  • Sales POS Opening Mode is Start/Sales by Menu Profile

First Come-First Serve (Drive-thru) - Guests queue up at order station where they place their order and move in the queue to the pickup station to get their order and pay.

  • Layout View is Order List
  • Order ID is Queue Counter or Receipt No. Only
  • After Transaction Posted is Stay in Sales POS
  • Sales POS Opening Mode is Start/Sales by Menu Profile

Delivery&Takeout in Call Center - Call center service for delivery & takeout in multiple restaurants.

  • Requires a sales type that is set as Delivery, Takeout or Pre-order sales type in Hospitality Setup
  • Layout View is Order List
  • Order ID is Delivery&Takeout Contact Name
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales

Delivery&Takeout in Restaurant - Restaurant delivery & takeout service, with possible connection to a call center.

  • Requires a sales type that is set as Delivery, Takeout or Pre-order sales type in Hospitality Setup
  • Layout View for Takeout and Pre-orders is Order List, for Delivery it is Delivery
  • Order ID is Delivery&Takeout Contact Name
  • After Transaction Posted is Return to Startup POS
  • Sales POS Opening Mode is Sales

Loading Card Station - A food station or cashier station using loading cards that are cards given to the customers to identify his or her order on multiple stations and at the cashier stations when exiting.

  • Layout View is Order List
  • Order ID is Loading Card No.
  • Order Guest Marking is On Combining Loading Cards Guest No. Specified
