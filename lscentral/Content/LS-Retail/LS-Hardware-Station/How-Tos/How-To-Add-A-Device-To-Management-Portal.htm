How to: Add a Device to Management Portal

To add a device to the LS Hardware Profile and LS Central

  1. Open the LS Hardware Station Management Portal in a browser (default location is localhost:8088) and log in.
  2. In the Device List, expand the tab for the device type that is being created (Printer, EFT, Cash Drawer, and so on), and click the Create button.
  3. In the pop-up window, a device ID has been automatically created. You can change the ID,if necessary.
  4. Select the OPOS device driver that has previously been installed on the computer.
  5. Click Create device, then select the device from the device list, and click Load.
  6. In LS Central, open the POS Hardware Profile List.
  7. Select the Hardware profile the device should be added to, and double-click to open the POS Hardware Profile Card.
  8. Click the Detect Hardware action.
  9. The device that was configured in previous steps is now visible in the Hardware profile in the appropriate list (Printer, EFT, Cash Drawer, and so on).
    Tip: Scroll down the POS Hardware Profile page to view the list.

Note: The Device Name corresponds to the device's logical device name (LDN) which comes from the registry. This can only be chosen from a list and should never be changed manually.

Note: The Device ID can be set by the user to anything he or she likes. The name will be in capital letters and should not be longer than 20 characters.

Note: After a device has been loaded, it will always be loaded when the Hardware Station is restarted, unless the device is deleted using the Delete Device button, or if the Ignore Device check box is selected.

To attach an existing device to a Hardware Profile

  1. In LS Central, go to the card for the device you want to add, for example POS Printer List, POS EFT List, POS Cash Drawer List, and so on. In these cards you see all devices that have either been detected by the LS Hardware Station or been manually added.
  2. If you want to create a device manually, the unique ID given to the device in the device card must be the same as the one given to the device in the Management Portal.
  3. To attach an existing device to a new Hardware Profile, go to the POS Hardware Profile Card, and make sure that the card is in edit mode before starting.
  4. Scroll down to the relevant FastTab, and select the device ID from the list.
