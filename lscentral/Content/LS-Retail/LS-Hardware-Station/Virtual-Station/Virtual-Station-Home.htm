How to: Use the Virtual Station

When the LS Hardware Station is installed, the Virtual Station is installed by default as well. The Virtual Station is very useful for testing and demonstration purposes. Note: Not all devices that are available in the LS Hardware Station are represented in the Virtual Station.

When you start the Virtual Station, it is very important to select Run as administrator, otherwise the LS Hardware Station cannot communicate with it.

Click here for instructions on how to install the LS Hardware Station.
Click here to view a video tutorial on how to install the LS Hardware Station.
To add the virtual devices to the LS Management Portal, a configuration called Load Virtual Station in the Server configuration needs to be activated. When the Server configuration is loaded again, the virtual devices will be created automatically.

To be able to use virtual devices in the POS, you must add them to the Hardware Profile through the Detect Hardware operation.
