How to: Create Group Infocodes

A group infocode is an infocode that groups together other infocodes.

You must use a POS Pop-up window. The POS Pop-up window displays the selection possibilities for one infocode in the group at a time. You can select any infocode to see its selection possibilities.

To create group infocodes

  1. Click the icon, enter Group Infocodes List, and select the relevant link.
  2. Click the New action to create a new group infocode.
  3. For filling in the fields on the page, refer to Setup Options for Group Infocodes.
  4. On the Group Selection Subcodes FastTab, create a new line.
  5. Enter a code in the Subcode field (01 for example).
  6. In the Infocode field, select the infocode you want to have in the group.
  7. Repeat for each infocode you want in the group.

Example:

Below you see a group infocode, QUESTIONS1, that groups together four infocodes, Gender, Age group, Flight number, and Travel Reasons.

  • Each individual infocode controls the input restrictions (maximum and minimum) for its input (gender is required (1 min, 1 max), age group is not required (1 max), flight number is required (1 min, 1 max), and travel reasons are not required, multiple selection allowed (each code only once)).
  • This group infocode does not set a limit on how many overall codes can be selected.
  • When you click OK, the system displays the next required infocode (OK Pressed Action).
  • Triggering is Automatic.
