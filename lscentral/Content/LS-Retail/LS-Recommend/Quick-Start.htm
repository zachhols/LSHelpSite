Quick Start

This is a brief step-by-step guide to get started with LS Recommend. The objective is to go through the necessary steps to create and build your first recommendation model.

Task 1: Subscribing to LS Recommend

Task 2: Inserting LS Recommend default data

Task 3: LS Recommend setup in LS Central

Task 4: Template for models and builds

Task 5: Create model and build to get recommendations

Next tasks

Task 1: Subscribing to LS Recommend

Prior to setting up LS Recommend in LS Central, you must subscribe to LS Recommend via your Partner or LS Retail Partner Operation. As soon as you have signed up, you will receive the necessary account information for setting up the module in LS Central.

Task 2: Inserting LS Recommend default data

In this task the subscription information obtained in the previous task is used to set up LS Recommend in LS Central.

  1. Click the icon, enter Retail Setup, and select the relevant link.
  2. Click the Insert Default Data action.
  3. Select the LS Recommend Data check box. This is to insert POS commands and default data used by the LS Recommend module.
    Note: If you only want to insert LS Recommend data, then click Clear All prior to selecting LS Recommend Data.
  4. Click Insert.

Task 3: LS Recommend setup in LS Central

In this task, the subscription information obtained in Task 1 is used to set up LS Recommend in LS Central.

  1. Click the icon, enter LS Recommend Setup, and select the relevant link.
  2. On the Account Info FastTab:
    1. Enter the admin key from Task 1 in the LS Account Key field, and make sure the Only Recommendation check box is not selected.
      Note: Admin rights are needed for the upcoming tasks.
    2. Enter End Point URI and Storage Account Connection with keys from Task 1.
    3. Click the Validate Key action to validate your setup.
  3. On the Usage FastTab:
    1. Enter a path to an existing directory in the Usage File Location field, or create a folder called TEMP on your local C drive.
      Note: Use \\ for \ when you enter the path.
  4. Go to the Display on POS FastTab, if you want to display the recommendation on a POS.
    1. Select the Use LS Recommend on POS check box.
    2. Specify how to trigger a call to the recommendation model on POS by selecting at least one of the following check boxes: Manual Display on POS, Print Recommendation on Slip, or Show Recommendation on Total.
    3. Select what information to show on the POS Panel that displays the recommendation results by selecting at least one of the following check boxes: Show Item Image, Show Item Description, Show Item Price, or Show Item Stock.
  5. Go to the Display in Omni FastTab, if you want to display the recommendation in Commerce.
    1. Select the Use LS Recommend in Omni check box.
    2. Set the Omni Func. Profile ID to the functionality profile that contains the connection information to Commerce.

Task 4: Template for models and builds

In this task default values for models and build procedures are selected. Note: Default values are provided by LS Recommend. If you are happy with using them you can skip this task.

  1. Click the icon, enter Model Template, and select the relevant link.
  2. On the Model Template page decide on default values, such as selecting an item catalog and what kind of usage data to use in the model and build.

Task 5: Create model and build to get recommendations

In this task a recommendation model is created with default settings.

  1. Click the icon, enter Models, and select the relevant link.
  2. Click the New action to create a model.
  3. Insert a name in the Description field, and click Create & Build Model with Data.
  4. View Builds, and click Set as Active Build when the build status is Completed.
  5. Click Enable to start using the model for recommendations.
  6. Depending on your setup:
    • POS: Go to Run Web POS, and log on to your POS to start using LS Recommend. Note: If you selected Manual Display on POS in LS Recommend Setup in Task 3, you must set the POS command RECOMMEND and select a parameter on a button.
    • Omni: Go to the Models page, and click Actions - Omni - Send Setup to Omni Server. Start using LS Recommend on Commerce Mobile POS.

Next tasks

The LS Recommend module aims to make product recommendations fast and easy to use, but it still allows advanced tuning of the underlying model.

The model created by first attempt may suit the data well and give excellent recommendations. To learn more about how to tune the model and/or improve default settings, see the options listed in the To-See table in the LS Recommend topic.
