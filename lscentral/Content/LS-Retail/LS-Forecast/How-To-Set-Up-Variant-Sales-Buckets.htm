How to: Set Up Variant Sales Buckets

The Variant Sales Buckets functionality lets you define the distribution portion of the variants sales for an item or a group of items.

The distribution portion or percentages are derived from weights that can be either assigned manually or calculated by the system, based on sales history.

To set up a Variant Sales Bucket

  1. Click the icon, enter Variant Sales Buckets, and select the relevant link.
  2. Click New to create a new Variant Sales Bucket.
  3. Fill in the relevant fields:
Field Description
Item Category Code This field specifies the Item Category to be considered for the variant sales bucket calculation.
Retail Product Code This field specifies the Retail Product Group to be considered for the variant sales bucket calculation.
Item No. This field specifies the Item to be considered for the variant sales bucket calculation.
Location Code This field specifies the location to be considered for the variant sales bucket calculation.
Per Location This check box specifies if the variant sales bucket will be calculated for each location in the group. This is only applicable to locations existing in the LS Forecast Location. If the check box is not selected and Location Code is blank, the calculation will consider the aggregated sales from all locations.

Note: You can only select this check box, if the Location Code is blank.
Sales History (Weeks) This field specifies how far back in weeks from the current WORKDATE should the sales history be considered for the variant sales bucket calculation. The sales history includes the records from both Value Entries and Sales History Adjustments.
Calculation Type This field specifies whether the sales history calculation will based on quantity or turnover.
  • Quantity - Quantity sold of the specified group.
  • Turnover - Sales amount of the specified group.

To calculate Variant Sales Bucket

  1. Click the icon, enter Variant Sales Buckets, and select the relevant link.
  2. Click the Calculate action to calculate all Variant Sales Buckets. Note: This will also delete all the existing Variant Sales Bucket Lines and Variant Sales Bucket Detail Lines.

The calculation is only performed for items with Replenishment Calculation Type set to LS Forecast, with LS Forecast Variant Grouping enabled, and with Variant Framework Code assigned. However, if an Item No. has been specified in the Variant Sales Bucket, it will be calculated regardless of the Replenishment Calculation Type. To include items with variant, but without Variant Framework Code assigned into the calculation, the Include Items without Variant Framework Code setting needs to be activated in LS Forecast Setup.

When a Variant Sales Bucket has been added, modified, or deleted, a recalculation is required to update the Variant Sales Bucket Lines and Variant Sales Bucket Detail Lines. It is possible to calculate the Variant Sales Bucket periodically by setting up a scheduler job assigned with the Codeunit Variant Sales Bucket Mgt. (10012212).

Note: LS Forecast Locations and LS Forecast Items should always be up-to-date before calculating the Variant Sales Bucket. By default, a notification is shown in the Variant Sales Buckets page as a reminder. This notification can be enabled or disabled via the Disable Notification (Variant Sales Bucket) setting in LS Forecast Setup.

To view the Variant Sales Bucket Lines

  1. Click the icon, enter Variant Sales Buckets, and select the relevant link.
  2. Click the Show All Lines action to view all Variant Sales Bucket Lines.
  3. Alternatively, select an existing Variant Sales Bucket, and click the Show Lines action to view only the Variant Sales Bucket Lines for the selected Variant Sales Bucket.

In the Variant Sales Bucket Lines page, the items included in the Variant Sales Bucket calculation are displayed. Selecting a Variant Sales Bucket Line will show the item's variant details including the calculated Weight and % Share in the Variant Sales Bucket Detail Lines FastTab. The Weight represents the quantity sold or turnover of the variants, depending on the Calculation Type specified for the related Variant Sales Bucket. The % Share represents the distribution portion based on the Weight. The Variant Sales Bucket Chart in the FactBox helps to visualize the distribution pattern of the variants.

Note: The Weight can be manually adjusted and the % Share will be recalculated automatically.
