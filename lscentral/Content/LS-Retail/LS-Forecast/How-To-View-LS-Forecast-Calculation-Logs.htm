How to: View LS Forecast Calculation Logs

To view LS Forecast Calculation Logs

To delete LS Forecast Calculation Logs

See also

Whenever a scheduler job is executed for LS Forecast, information such as the start and end date time for each task, number of forecast entries downloaded, and task status will be logged into the LS Forecast Calculation Log.

To view LS Forecast Calculation Logs

  1. Click the icon, enter LS Forecast Calculation Log, and select the relevant link.
  2. You can also view the LS Forecast Calculation Logs by clicking the Current Usage % tile under LS Forecast - Usage Monitoring in the LS Replenishment Role Center. For more information on calculation limit, see How to: Set Up LS Forecast Calculation Limit.
  3. Below are the fields in the LS Forecast Calculation Logs page:
Field Description
Scheduler Job ID This field shows the job that was scheduled for the master data synchronization or forecast calculation.
Batch No. This field shows the batch number of the forecast calculation.
Master Data Sync. Started This field shows the date and time when the master data synchronization started.
Master Data Sync. Ended This field shows the date and time when the master data synchronization ended.
Master Data Sync. Duration This field shows the elapsed time of the master data synchronization.
Calculate Forecast Run ID This field shows the run ID of the forecast calculation.
Calculate Forecast Started This field shows the date and time when the forecast calculation started.
Calculate Forecast Ended This field shows the date and time when the forecast calculation ended.
Calculate Forecast Duration This field shows the elapsed time of the forecast calculation.
No. of Calculations This field shows the number of calculations performed in the forecast calculation.
No. of Forecast Entries This field shows the number of forecast entries downloaded from the forecast calculation.
Calculation Blocked This field shows if the forecast calculation was blocked due to it exceeding the calculation limit.
Error Occurred This field shows if an error occurred during the master data synchronization or forecast calculation.
Message Text This field states the result of the scheduled job, or if the log is related to a forecast calculation run manually. Additionally, if a forecast calculation is exceeding the warning threshold or the calculation limit, that will be indicated in this field.

To delete LS Forecast Calculation Logs

The LS Forecast Calculation Logs can be deleted periodically by setting up the Scheduler Job DELETELOGS and the Days Calculation Logs Exists field in LS Forecast Setup.

  1. Click the icon, enter LS Forecast Setup, and select the relevant link.
  2. In the General FastTab, fill in the Days Calculation Logs Exists field. This field specifies how far back in days to keep the LS Forecast Calculation Logs. All LS Forecast Calculation Logs older than the defined period will be deleted, if the Scheduler Job DELETELOGS is enabled.
