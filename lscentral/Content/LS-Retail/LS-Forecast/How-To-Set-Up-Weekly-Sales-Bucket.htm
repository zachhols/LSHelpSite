How to: Set Up Weekly Sales Buckets

The Weekly Sales Buckets functionality lets you define the distribution portion of the days in the week for an item or a group of items.

The distribution portion or percentages are derived from weights that can be either assigned manually or calculated by the system, based on sales history.

To set up Weekly Sales Bucket

  1. Click the icon, enter Weekly Sales Buckets, and select the relevant link.
  2. Click New to create a new Weekly Sales Bucket.
  3. Fill in the relevant fields:
Field Description
Item Category Code This field specifies the Item Category to be considered for the weekly sales bucket calculation.
Retail Product Code This field specifies the Retail Product Group to be considered for the weekly sales bucket calculation.
Special Group Code This field specifies the Special Group to be considered for the weekly sales bucket calculation. It serves as an additional filter for items.
Item No. This field specifies the Item to be considered for the weekly sales bucket calculation.
Store Group Code This field specifies the Store Group to be considered for the weekly sales bucket calculation. It serves as an additional filter for locations.
Location Code This field specifies the location to be considered for the weekly sales bucket calculation.
Per Item This check box specifies if the weekly sales bucket will be calculated for each item in the group. Only items existing in the LS Forecast Item will be considered. If the check box is not selected and Item No. is blank, the calculation will consider the aggregated sales for the specified Item Category, Retail Product Group, or Special Group Code.

Note: You can only select this check box, if the Item No.is blank.
Per Location This check box specifies if the weekly sales bucket will be calculated for each location in the group. This is only applicable to locations existing in the LS Forecast Location. If the check box is not selected and Location Code is blank, the calculation will consider the aggregated sales from all locations.

Note: You can only select this check box, if the Location Code is blank.
Sales History (Weeks) This field specifies how far back in weeks from the current WORKDATE should the sales history be considered for the weekly sales bucket calculation. The sales history includes the records from both Value Entries and Sales History Adjustments.
No. of Lines This field specifies the number of calculated Weekly Sales Bucket Lines for the Weekly Sales Bucket. You can access the Weekly Sales Bucket Lines page by clicking this field.
Calculation Type This field specifies whether the sales history calculation will based on quantity or turnover.
  • Quantity - Quantity sold of the specified group.
  • Turnover - Sales amount of the specified group.
Priority This field specifies which Weekly Sales Bucket Line should be considered for breakdown, when there are multiple Weekly Sales Bucket Lines with the same Item No. and Location Code. The priority number starts with 1, which is the highest rank.

To calculate a Weekly Sales Bucket

  1. Click the icon, enter Weekly Sales Buckets, and select the relevant link.
  2. Click the Calculate action to calculate the selected Weekly Sales Bucket line.

It is possible to calculate the Weekly Sales Bucket periodically by setting up a scheduler job assigned with the Codeunit Weekly Sales Bucket Mgt. (10012228).

Note: LS Forecast Locations and LS Forecast Items should always be up-to-date before calculating the Weekly Sales Bucket. By default, a notification is shown in the Weekly Sales Buckets page as a reminder. This notification can be enabled or disabled via the Disable Notification (Weekly Sales Bucket) setting in LS Forecast Setup.

To view and modify Weekly Sales Bucket Lines

  1. Click the No. of Lines field on the Weekly Sales Bucket that you wish to view or modify.  
  2. The Weekly Sales Bucket Lines page is displayed with the calculated weight and percentage share. 

    Based on the Calculation Type selected, the Weight represents the total sales quantity or amount grouped by each day of the week across the period defined by Sales History (Weeks).

    The % Shares for the days of the week are automatically calculated and distributed, based on the Weights.

    Note: If there is no sales history found, the Weights and % Shares will be equally distributed.

  3. You can also modify the Weights manually, if needed.

To set filter on Weekly Sales Bucket Lines

  1. Click the Set Filter action on the Weekly Sales Bucket Lines that you wish to view or modify.
  2. A custom filter page is displayed where you can set or clear filters on the Weekly Sales Bucket Lines page, to get an overview of the calculated results and priority assigned.
