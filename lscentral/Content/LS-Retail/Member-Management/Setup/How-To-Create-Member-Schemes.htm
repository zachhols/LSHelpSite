How to: Create Member Schemes

Every member club has one or more member schemes. The schemes are used to split members into different groups based on purchase or point collection. It is possible to specify different benefits and rules for each scheme in the club. The system handles upgrades and downgrades between the schemes, depending on number of purchases or points earned. The rules for how this is handled are specified in each scheme.

You could, for example, create a member club with three member schemes: Bronze, Silver and Gold. You could then set the schemes up so that members start in the Bronze scheme and upgrade to Silver when they have earned enough points. There is no limitation of the number of schemes in the system and they can be created in two ways:

Note: Many of the scheme settings on the Member Schemes card are also available on the Member Club card. The Member Scheme settings will always have higher priority than the Member Club settings.

To create a new member scheme

  1. Click the icon, enter Member Schemes, and select the relevant link.
  2. Click the New action. The Member Scheme Card is displayed.
  3. In the Code field, enter a unique identifier for the scheme. It is not possible to have more than one scheme with the same code.
  4. Select the member club you wish to link to the scheme from the drop-down list in the Club Code field.
  5. Enter a description for the scheme in the Description field.
  6. Fill in the fields in the Upgrade FastTab as described in the following table.
    FieldDescription
    Update SequenceThis field specifies the sequence of updates between schemes. For example, you would insert 1 for Bronze, 2 for Silver, and 3 for Gold. A member account can be upgraded either one or two steps at a time: the scheme can be changed from Bronze to Gold and vice versa.
    Buffer Days Before ExpireThis field specifies the number of buffer days before the member points expire.
    Min. Point for UpgradeThis field specifies how many points you need in a given period to be upgraded from one scheme to another (for example from Bronze to Silver). The time frame is specified in the Period Calc for Up-/Downgrade field on the Member Club card, Point Calculation FastTab.
    Next Scheme BenefitsHere you can enter free text describing the benefits when upgrading to the next scheme.
    When Check for UpgradedThis field tells the system how frequently it should check member accounts for upgrade. The options are: Daily, Weekly, Monthly, Quarterly, and When Expire.
    Point Limit for DowngradeHere you specify the minimum number of points needed to stay in the scheme. If the customer collects less than the minimum required points in the period (specified on the Member Club card), he or she will be downgraded to the next scheme below, and even lower if he has fewer points than specified in the downgrade limit for that scheme.
  1. You can now open the Member Clubcard, and select the new scheme in the Default Scheme field in the General FastTab. This is the scheme that members will be automatically assigned to when they join the club.

Note: You can also create schemes on the Member Club card: press New, on the Member Schemes FastTab.

The next step is to create member accounts for the club. See How to: Create Member Accounts for more details.
