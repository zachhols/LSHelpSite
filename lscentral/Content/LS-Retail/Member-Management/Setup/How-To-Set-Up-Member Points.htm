How to: Set Up Membership Points

A rule for how membership points are calculated must exist in the system with at least one rule per member club. The rules define how many points will be awarded to the customer through sales transactions.

To set up member points

Points can be transferred from one account to another in the same club, and points can also be bought or transferred from other clubs.

Points are earned in different ways: for example, by sales transactions (amount or quantity), specific tenders, or by specific customers.

The quickest way to access the membership point setup is this one: 

  • Click the icon, enter Membership Point Setup, and select the relevant link.This page is an overview of the total setup for all clubs, schemes, and accounts.

You can also access the point setup on the following cards:

  • Member Club
  • Member Scheme
  • Member Account

The procedure to access the setup is basically the same on the first two cards: 

  1. Highlight a club/scheme in the list.
  2. Click Edit on the Home action menu.
  3. Open the Actions menu, and click Point Setup .

In the Member Accounts page you highlight the account and click Related - Account - Point Setup directly on the action menu.

To set up member points

  1. Open the Membership Point Setup card, and click the New action to create a new point setup.
  2. Fill in the fields. The Scheme Filter Type, Club/Scheme, and Unit Rate fields are required fields.

Note: It is not necessary to create a point setup for a club and its scheme. The system always looks first for point setup for a member account, and if such a setup does not exist, the system looks for a setup for the member scheme, and finally, the member club.
