How to: Create or Replace a Terminal

The easiest way to create a new terminal when adding a new POS is to use the Request New Terminal functionality in the POS Terminal Card. To be able to request a new terminal from POS, you must configure the Head Office correctly, replicate from Head Office to POS, and then do a Request New Terminal from Head Office request from the POS.

Head Office setup

POS setup

Users will be prompted to select a terminal template which the new POS Terminal will be created after. The template has to exist in both the POS and the Head Office.

  • Use Case 1: Store owner is setting up a new POS. He or she is able to install the LS Central on the POS, replicate to the POS and create a new POS terminal for the POS with a few clicks on the POS.
  • Use Case 2: Store owner needs to replace a POS Terminal already running LS Central. He or she is able to request a new POS Terminal with a few clicks.

Head Office setup

1 Create Terminal Template

Terminal Templates are terminals that are copied when the POS requests a new terminal.

  1. To create a new Terminal Template, open the POS Terminal List and click New.
  2. In the No. field, insert your desired template name, for example #ENT. The terminal template No. has to start with # and cannot be longer than 10 characters. The Store No. can be any value since it is changed each time a new terminal request is made.
  3. In the Functionality Profile field, click the down arrow and select a profile to use.

2 Set Distribution Location for Functionality Profiles

The POS Functionality Profile must have a Distribution Location set for the POS to know what terminal it should get.

  1. Go to the POS Functionality Profile List, and click Manage - Edit for a profile that is used by a Terminal Template. In this example the ##Default profile will be edited.
  2. The POS Functionality Profile Card opens. Click the Profile action, then click Web Servers.
  3. Make sure that the Local Requests check box is not selected, and select your Head Office in the Dist. Location field.

3 Set Web Service URI in Distribution Locations

Web Service URI is necessary for the POSs to communicate with the Head Office. This is also needed for replication data from HO to POSs.

POS setup

Request new terminal in Business Central

To request a new POS terminal in Business Central, open the POS Terminal List, click Actions, and then click Functions - Request New Terminal from Head Office. This will open a new page where you can specify which store the new terminal should belong to, what Terminal Template to use, and where you can enter a short description.

Request new Terminal from LS Start

  1. Open LS Central Start.
  2. Create a new button or edit an existing one by right-clicking a button and selecting the appropriate option.
  3. Configure the button, and then press the Request a New Terminal button.
  4. Click Save, and then click the LS Central Start button. A Business Central form will prompt the user to select which store to create the POS Terminal in and which POS Terminal Template to use. A new Terminal will then be created for the POS.
