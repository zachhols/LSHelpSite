Retail Images can be set up for Menus, Retail Items, Variants, Retail Product Groups, Item Categories, Published Offers, Stores, Media Controls, Media Playlists, Search Index, Member Notifications, Member Contacts, Membership Cards, Retail Setup, Hierarchies, Staff Management Employees, and Dining Table properties.

  • Go to the Retail Image List to add new or edit existing images.
  • Retail Images can be linked to the entities above from the respective list or card pages, see below.

The Retail Image Card

Here you can import new Retail Images and remove existing ones. Click Picture in the FactBox to the right to see the available actions. Use the Where-Used action to find out where the image is used.

Fields in the General FastTab

Field Description
Code The code for the image is used to link an item, store, offer, and so on to the image.

To link an image to an item, go to the Retail Item Card, and click Related - Master Data - Images. This opens the Retail Image Link List where you insert the Code of the image in the Picture ID field.
Image Location This field is used for links to outside of the LS Central database. If you place a link in this field, the link is followed to show the link's image and the Retail Image is not shown. So in this case no image should be imported.
Description Description of the image.

The Retail Image Link List

To access this list, go to the Retail Item Card, and click Related - Master Data - Images to open the Retail Image Link List page. Here existing Retail Images are linked to the record they are activated from.

Fields

Field Description
Image Id Here you enter the code for the Retail Item.
Link Type This field is used for different types of images. Possible values:
  • Image: Images for normal use.
  • Logo: Images for store logos.
  • QR Code: Images for QR code for directions to the store.
Display Order If more than one image is stored for an entity, the image with the lowest value is displayed. Note: Images are ordered by Link Type.

Assigning an image to a menu button

You can link a Retail Image to POS Menus, both header and lines.

For the header:

  • In the POS Menu, click Actions - Images to open the Retail Image Link List.

For the lines:

The Image column tells you if there is an image linked to the line or not.

  • To open the Retail Image Link List, click the Image column in the relevant line, or click the Manage action in the Lines FastTab, and then click the Images action.
