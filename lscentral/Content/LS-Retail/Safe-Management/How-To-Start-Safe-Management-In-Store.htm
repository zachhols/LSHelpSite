How to: Start Safe Management in a Store

Start working with Safe Management by preparing a store.

1 To prepare a store

2 To link POS terminals to the Store safe

3 To link Retail Users to the POS terminals

1 To prepare a store

  1. Click the icon, enter Store List, and select the relevant link. Create or select a store.
    1. Existing store
      1. Calculate and post statements for all POSes or Staff for the store depending on how the statements are for the store.
      2. Make sure that no entries exist for POSes or Staff in the POS Start Status and POS Start Entry tables.
    2. New store
      1. Create a new store and POS terminals.

  2. Click the icon, enter Store Safe List, and create a store safe, and link the safe to the store by selecting that store from the Store No. drop-down list.

  3. Click the icon, enter Store List, and select the relevant link.
  4. Select a store from the list.
  5. On the Statement/Closing FastTab:
    1.  Select the Safe Mgmt. in Use check box.
    2. Set the POS Start Amount Method field to Fixed Bag.
    3. Set the Statement Method field to POS Terminal.
    4. Enter POS Fixed Start Amounts:
      1. Click the Navigate action, and then click Store - Cash Management - Fixed Start Amounts.
      2. Enter Fixed Start Amount for each POS and denominations, if needed.

    5. Select the POS Post Safe Ledger check box.
      Note: Do not select this check box if the POSes are working in their own database and sending the sales transactions to a central database, either in the store or at the Head Office.

    6. Configure Tender Types:
      1. Click the Navigate action, then click Store - Cash Management - Tender Types .
      2. Select a tender type, and for each one (normally only Cash) do the following:
      3. On the General FastTab, select the May Be Used check box.
      4. On the Declaration FastTab, select all check boxes except To Safe by Denom..
        Note: Do not select Count By Denomination on the Posting FastTab.

2 To link POS terminals to the Store safe

  1. On the POS Terminal Card select a POS in the Safe No. field.
  2. Create Start Float for POS Terminal:
    1. Click the icon, enter Safe Transfer List, and select the relevant link.
    2. Click the New action.
    3. Leave the Transfer No. field empty (automatically filled from a Number Series).
    4. Select the safe in the Safe No. field.
    5. In the Transfer Direction field, select To Safe.
    6. Select Bank Account to retrieve money from the bank.
    7. Transaction Line:
      1. In the Safe Type field, select Fixed Float.
      2. In the Statement Code field, select your POS Terminal Number.
      3. Select a tender type for cash.
      4. Enter a description in the Description field.
      5. On the Lines FastTab, click the Manage action and then Bag.

        The bag's content is displayed.

      6. Click the Only Total action (or By Denomination and enter coins and notes by denomination).
      7. Enter Total Amount (for example 1500 Euros).
      8. Post the transfer.

3 To link Retail Users to the POS terminals

  1. Click the icon, enter Retail Users, and select the relevant link.
  2. Click the Edit List action.
  3. Enter POS Terminal numbers for users (and the Store No., if missing).
