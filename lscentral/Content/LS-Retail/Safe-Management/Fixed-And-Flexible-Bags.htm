Fixed and Flexible Bags

To set the POS Start Amount Method

  1. Click the icon, enter Stores, and select the relevant link.
  2. Select a store to open the Store Card, and go to the Statement/Closing FastTab.
  3. The options in the POS Start Amount Method field are:
  • Flexible - the POS does not start the day with Insert Float since the money is already in the safe. The money for the drawer that is taken to the POS is counted via Add Float.
  • Fixed Bag - the first time the POS is started, the start amount defined by denomination on the Store Card is put in the drawer at the beginning of the day. After that, the starting amount for each day is what was in the drawer the last time the POS was closed.
  • Flexible Bag - the POS is opened when the amount by denomination is 0 for each type of coin or note. The user manually inserts the amount for each. A bag of floats for the POS is created or, if it already exists, a “Total” bag is used.

If you select the Fixed Bag option, a Fixed Bag is initially taken from the bank or booked by the user through a G/L account to the safe. This bag exists in the safe at the beginning and end of each day. When the POS is opened, the bag is automatically moved to the drawer in the system to be used during the day.

Note: To be able to use the Fixed Float feature, you must first select the Fixed Float check box on the Declaration FastTab on the Tender Type card (see here for more details).

To view the fixed start amount on the Fixed Start Amounts card 

  1. Click the icon, enter Stores, and select the relevant link.
  2. Double-click a store to open the Store Card.
  3. Click the Navigate action, and then click Store - Cash & Safe Management - Fixed Start Amounts.

The POS Fixed Start Amounts card displays information about the tender type and the Fixed Start Amount.

You can enter the cash in the safe by Denomination. Once the Fixed Float has initially been entered to the safe, this amount will be available at the start of the day. The cash can change from day to day and then the difference is booked.

Note: You can always check whether the correct amount is in the safe (it may be more than just the start amount). This is done by viewing the Store Safe card and by looking at the Fixed Float Bag and the combination of the money by clicking the number in the Balance field. The Safe Ledger Entry list opens and there you will find the Fixed Float amount.
