Safes

Each store can have as many safes as needed. This topic describes how to create and delete safes. You can create a new safe at any time.

To create a safe

  1. Click the icon, enter Store Safe List, and select the relevant link.
  2. Click the New action to create a new safe.
  3. Fill in the fields on the Store Safe page.

    The fields on the General FastTab are:

    • Safe No.: Name or number of the safe.
    • Description: A description of the safe, if necessary. For example Upstairs or Downstairs safe.
    • Store No.: You select the identification number of the store where the safe is located from the list.
    • Store Name: The name of the store where the safe is located.
    • Balance: The amount stored in the safe at any time. For example, money is often transferred initially from the bank to the safe to use as a Fixed Float. The amount is displayed as a balance.
    • Balance LCY: The total amount in local currency stored in the safe.

    Note: When the Fixed Float is taken from the safe and put into the POS while the POS is open, the Balance and Balance LCY are shown as 0,00. When the Fixed Float is returned to the safe after the Tender Declaration and End of Day procedure, the same amount should be returned to the safe for the Fixed Float, plus the amount that has been put into the safe. This is, for example, the amount kept in the safe overnight before it is sent to the bank in the morning.

  4. Next, fill in the fields on the Setup FastTab on the Store Safe card.

    • Transfer No. Series: A series used for numbering transactions in Safe Management.
    • New Bag No. Series: A series used for numbering bags in Safe Management.

To delete a safe

Certain rules apply when a safe is to be deleted. A safe can be deleted as long as there is no money in the safe and there are no entries connected to it. If any of the following entries are connected to the safe, the safe will not be deleted:

  • Safe Ledger Entries
  • Posted Safe Transfers Entries
  • Safe Bag Entries
  • Safe Journal Line Entries
  • Safe Statement Line Entries
  • Joined Safe Bag Entries
  • Safe Transfers Header Entries

If none of these entries are connected to the safe, you can delete the safe:

  1. On the Store Safe List, select the safe you wish to delete.
  2. Click the icon to delete the safe.
  3. If it can be deleted, you are asked to confirm that you want to delete the safe. Click Yes to confirm.

Note: If there are still some entries linked to the safe that you want to delete, an error message appears and you cannot continue. Click Yes to discard the latest changes (that is your request to delete the safe), and stop the deleting process.
