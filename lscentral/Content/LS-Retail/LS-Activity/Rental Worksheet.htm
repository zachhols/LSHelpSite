Rental Worksheet

The Business Central back end rental worksheet is used to assign rental units or return rental units to a selected activity line (rental charge). You can access this worksheet from the Reservations card by using the Unit Tracking action, from reservation lines, or from the Activity Card.

Note: The Show Rental Units check box must be selected on the related reservation type.

Note: The assignment feature is only available on the Reservation Card Lines or Activity Card using the Unit Tracking option on the menu section in those pages. If you access the worksheet from the Reservations card, keep in mind that no rental units can be assigned here and only returns can be registered. This is because that when a rental is assigned, the Activity Charge line, which is the basis of the rental, must be selected first and that cannot be done on the Reservations card.

The worksheet view shows the main section as the units are being assigned and/or returned during the entry process, but the Confirmed Entries section shows those entries after they have been confirmed and processed.

While the lines are visible in the worksheet section, they have not yet been processed and are in Draft status.

When you enter the Unit No. the entry is validated based on the entry type. If the entry type is the default type, the system assumes that you are assigning the unit (if it is available). If the unit is already in rental by this particular client, the system assumes that the entry is a return entry. You can also set the entry type to be a fixed assignment or a return only, which might be a more secure method in some circumstances. Unit validation also makes sure that the rental unit is available when it is assigned, and it also makes sure that the unit is actually in rental when it is being returned.

If you need to exchange units during the rental, you can simply return the first unit (which should at that point be registered in rental), and assign the new unit to the rental contract (reservation).

Unit Tracking Worksheet sections

Section Description
Unit No. Here you can enter the unit number to be assigned or returned. If the unit has already been entered into the worksheet lines, it is removed (toggle effect).
Product Resources FactBox Shows which resources (rentals) are accepted related to the currently active rental charge (activity) and how many have been assigned already. For example a rental charge for bicycle will only allow the user to scan rental units which are bicycles.
Attributes FactBox Shows the attributes of the rental unit in the worksheet line.

The Actions menu

Action Description
Confirm Confirms (validates) the worksheet lines and applies the entries to confirmed entries. Worksheet lines with an error message are not confirmed.
Assignment Sets the Entry Type Method to Assignment only. By default, the system determines by the rental unit's current status whether it is being assigned or returned, but the user can force the entry method to be assignment only.
Returning Sets the Entry Type Method to Returning only.
