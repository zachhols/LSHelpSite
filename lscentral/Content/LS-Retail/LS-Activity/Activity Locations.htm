Activity Locations

The Activity Locations page lets you specify setup options for locations.

  • Click the New action to create a new location.
Column Description
Code A unique code reference for each location.

Note: You must set up an activity location even if you will only be using a single location. This will make your organizational data ready for future growth and when you add new locations this is the only additional setup required.

Description The name or description of the location.
Store The retail store which relates to this location. Only used to retrieve opening hours.
Opening Hours The available options are Location Settings or Store Settings. Assign if opening hours are manually entered for the location and maintained in the Activity module, or if opening hours should be retrieved from the retail settings.
Open From / Open To The default opening hours (if Location Settings is being used).
Note: If Open To starts before Open From, it is assumed as an over-midnight opening.
Work Location The Staff Management work location for the same activity location. Only used for those employees where the solution needs to retrieve availability based on their roster assignment. If you leave this field blank, no Staff Management integration is used.
Dining Area Only used for table reservation integration with LS Hospitality.
  • Click the Opening Hours action to open the Activity Location Opening Hour page where you have access to detailed settings of opening hours per weekday or for a specific date.
  • Leave the Time From and Time To fields blank to indicate that the location is closed on that particular day or date.
