Additional Charges

Each activity product can have multiple additional charges and consumption registered.

The charges are automatically added when the product is selected. Some charges may be optional, and the user can select or deselect, depending on the additional charge settings for the product.

The table below lists and describes the fields on the Activity Card's Additional Charges FastTab:

Field Description
Select A shortcut to select the current charge line and set Quantity to default quantity, if it is zero.
Product No. The product number of the charge that is added to the activity.
Description Normally, the description of the product. If, however, the product number is left blank, then this field can be edited to serve as a title or a comment.
Quantity The quantity to be added.
Unit of Measure The unit of measure of the product, which then affects pricing and stock quantity.
Price The unit price of the additional charge, populated by the product price.
Discount % The assigned discount. Note: The user may have restrictions on the value of the allowed discount.
Total Amount The total net amount to be charged to the customer.
Quantity Multiplier If the quantity is being calculated based on the quantity set in the Activity field, the No. of Persons field, or both. This is decided by the setting on the activity charges on the activity product. For added lines, the default is No linking to the activity quantity.
Entry Type Is either By charge or Consumption. Is a non-editable field.
Status By default this field is blank and the line is editable. But when charges have been posted, the status changes to Posted, and the line becomes non-editable.
Availability This field shows the current available inventory for the additional charge product. If the product is a normal retail item, the field shows the stock inventory at the related location. If the additional charge has resource dependency, the availability of resources is shown. If there are resources required for this additional charge, this field will also change to indicate that resources are reserved.
