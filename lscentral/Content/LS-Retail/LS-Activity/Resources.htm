Resources

Activity resources are resources that are needed to perform the activity products (services) being offered and sold. The resources must be assigned to a resource group, which by default defines the various aspects and availability of the resource.

The Role Center's Resources section

To create an activity resource

To view the Resource status

There are two ways to access the Activity Resource Card where you work with resources:

  • through the Resources section of the Role Center or
  • from the Departments menu.

The Role Center's Resources section

The Resources section of the Role Center (on the right-hand side) shows all currently active activity resources for the selected date and location, their current availability and assigned activities with drill-downs to the details. You can phone or email directly by clicking the Phone No. or Email fields.

To open the Activity Resource Card

  1. Click Resources in the Role Center's left navigation pane.
  2. Click New or Edit on the Home action menu, or double-click an item in the Resources list to display the Activity Resource Card.

To create an activity resource

  1. Click the icon, enter Activity Resources, and select the relevant link.
  2. Click the New action menu to display the Activity Resource Card.

Here is a shortcut to a description of the card's FastTabs and fields:

  • Group FastTab
  • Availability FastTab
  • Contact FastTab

Click here to see descriptions of the actions on the Activity Resource Card.

Group FastTab

Field Description
Resource Group The resource group to which the resource is assigned and where the resource picks up default settings.
Resource No. The unique ID used to refer to the resource.

Note: When assigning human employees as resources, the user can here look up in the list of existing employees for an employee tp assign as a resource.
Description The description or name of the resource.
Status Specifies if the resource is active and included in availability.
Gender Only valid for human resources.

Availability FastTab

Field Description
Availability Quantity Fixed Availability (only for resources with fixed quantity of availability).
Fixed Location Assign a location if the resource is available in one activity location only.
Capacity Based On The definition of the availability logic. See Activity Resource Groups for detailed information.

Here the user can change the availability logic to differ from the default setting for a single resource. If you select Default, then the availability logic derives from the resource group setting.
Selection Priority The default selection priority (0-10) that is assigned on the resource for automatic resource selection during assignments. This value is used for sorting purposes.

Note: The resource can also have the selection priority set differently on specific products in the capability registration to give higher or lower priority for different products, if needed.
Cost Price Type Define if the cost price on the Resource card should be assigned on each resource reservation per activity, or per hour. It is also possible to specify cost price per Activity Product for the resource, through the Resource Capabilities settings. Those settings are always per Activity, and the Per Hour is only used as basis for calculation if no specific price is set in the Capabilities settings.
Cost Price The cost price to be used for any assignments of the resource, either per hour or per activity as set by the Cost Price Type. The cost prices can be summarized by the Summary Views as defined by the Summary Setup.

Contact FastTab

Field Description
Phone No. The phone number of the resource, if applicable.
E-Mail The email of the resource, if applicable.
Staff ID Links the resource to staff or salesperson in LS Central. This will ensure that the resource is assigned as a salesperson on the activity when related activities are finalized through the POS. If the LS Central Sales commission functionality is used, then this would be a required setting to enforce resource assignment (salesperson).

To view the Resource status

  1. Select a line in the Resources list (Role Center), or in the Activity Resources list (Departments).
  2. Click Resource Status on the Home action menu to display the Activity Resource Status view.

The view has the following columns and actions:

Columns Description
Entries Shows all activity entries that day or starting at the specific time.
Tasks Shows all tasks that day or those starting at the specific time.
Drill-down Drill-down from the resource matrix cell will list any assigned activities.

 

Actions Description
Attribute Search To search and limit the Resource View according to the search template. The template can define which attributes are required.
Transfer Activities To transfer activity assignment from one resource to another or multiple others.
Day Availability Shows resource status as one line per day and the availability of the resource.
