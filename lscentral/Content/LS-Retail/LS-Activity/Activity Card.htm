Activities

The Activity Card shows a single activity, which is basically a single activity line in a reservation.

It is important to understand that the Activity Card is simply a more focused view on a single activity. The card, however, presents (and contains) exactly the same information as the activity lines on the Reservation Card.

Some sections of the Activity Card may differ, depending on what activity product is being edited, since the activity type (and the reservation type) dictate what sections and fields are enabled for each product.

To edit the Activity Card from the Role Center

  1. Click Activities in the left navigation pane.
  2. Double-click an entry in the list to display the Activities Card, or select an entry and click Card on the Home action menu.

The tables below list and describe the fields on the Activity Card's FastTabs. Here are shortcuts to the descriptions:

  • General FastTab
  • Client FastTab
  • Period FastTab
  • Product FastTab
  • Additional Charges FastTab

To see descriptions of the actions on the Activity Card, click here.

General FastTab

Field Description
Activity type The activity type.
Comment Internal free text comment.

Client FastTab

Field Description
Client No. The client's contact number in the Member Management module.
Client Name The name of the client.
Mobile Phone No. The client's mobile phone number. Note: If Client No. is blank, then entering a phone number will trigger a search in the member contact records to retrieve customer details based on the mobile phone number.
E-mail The client's email address. Note: If Client No. is blank, then entering an email address will trigger a search in the member contact records to retrieve customer details based on the email address.

Period FastTab

Field Description
Date From The starting date of the activity.
Time (From) The starting time of the activity.
Time (To) The ending time of the activity.
Date To The ending date of the activity.
Duration The duration of the activity (optional).
Duration Type The type of duration, None, Minute(s), Hour(s), or Day(s).

Product FastTab

Field Description
Product No. The activity product's number.
Description The activity product's description.
Quantity The quantity sold. Quantity may be converted automatically, based on the duration setting on the activity product. Note: Changing the quantity manually may affect duration, depending on product settings.
Unit The quantity unit, that is None, Day(s), Hour(s), Minute(s), Week(s), Month(s), Year(s).
Unit Price The price of each unit.
Discount % The line discount percentage. Either manually entered or by promotions.
Discount Amount The discount as amount.
Amount The total amount to be charged for the activity product.
No. of Persons The number of persons assigned on the activity (optional).
Unit No. Only valid for rental products. A shortcut field where the user can assign or return rental units without opening the unit tracking worksheet.
Promo Code If promo code is entered, then its validated against active promo codes and then applied for discount or pricing settings.

Additional Charges FastTab

This FastTab shows any additional charges assigned to the activity. These charges can be setup as a template on the activity product, and are then automatically populated.

Also, the FastTab shows the consumption entries, which are not charged, but only posted as stock adjustment to the item ledger.

Note: Any retail offers and promotions are never affecting these items. Any retail promotions on the activity products will be calculated when the activity is paid on the POS and will update the activity after posting the receipt.

Field Description
Select If the entry is selected and added as charge. The user can toggle this setting on or off for any items that are set up as „optional” additional charges in the template.

If this field is not enabled, the item has been defined as not being optional and is always added to the charges.
Item No. The retail item number.
Description The retail item description.
Quantity The quantity assigned.
Price The price of the item.
Discount % Manual discount percentage.
Total Amount Total amount added to the activity charge.
Quantity Multiplier Defined by the Activity product template for additional charges. Specifies if the quantity of the retail item line should be multiplied by the quantity or the number of persons on the activity charge.
Entry Type Shows if the line is an additional charge or a consumption entry.
Status Shows if charge entries have been paid or if consumption entries have been posted.
Availability Shows the current available inventory for the additional charge product. If the product is a normal retail item, the field shows the stock inventory at the related location. If the additional charge has resource dependency, the availability of resources is shown. If there are resources required for this additional charge, this field will also change to indicate that resources are reserved.
