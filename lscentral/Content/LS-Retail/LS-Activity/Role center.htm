Role Center

The Role Center is, as the name suggests, a central view of all activity in the current location for the current date.

You can change the date by changing the Business Central work date. See the Business Central help for more details.

Cues

The below table lists the available cues in the Role Center:

Cue Description
Today’s Activities  All activities that are starting at the current date.
Unpaid Activities All confirmed and closed activities at and prior to the current date which have not been finalized.
Draft Reservations All reservations with the internal status Draft.
New Activities All new activities created on the current date.
Canceled Activities All activities which have been canceled on the current date.
Active Promotions All prices and discounts that have been marked as promotions.
Admission Tracking Overview of all admission arrival logs for the current date.
Tasks Shows all open tasks that are due on the current date.
Pickups All activities which include rentals and are starting today.
Returns All activities which include rentals and are ending today.
Overdue All activities which include rentals and have ended, but rental units have not been returned.
Maintenance Due All rental units due for maintenance, based on their usage and maintenance cycle settings.
Arriving Groups All activity groups which are arriving. By default the reference date is WORKDATE. You can, however, specify in the Activity Setup if the Arrival Cue also shows groups arriving within X number of days.
Current Groups All activity groups which are currently confirmed at the current WORKDATE.
Departing Groups All activity groups that are departing. By default the reference date is WORKDATE. You can, however, specify in the Activity Setup if the Departure Cue also shows groups departing within X number of days.

Insights Sections

The Insights part of the role center contains 3 views (or parts) which each display vital information on the current state of activities, products, and resources:

  • Activity Timeline - always shows TODAY's or WORKDATE activities in chronological order. From the action menu you can drill down to the relevant Activity Card, Resource Card, Client Card, or Reservation Card details.
  • Resources - shows all active resources, their current status, and the number of activities assigned. From the list you also drill down to the related activities.
  • Products - shows all active products and pricing information. From the action menu you can change the order sequence of the list, and whether the list is based on activity type or resources. You can also view the details of the product, that is images and attributes, as well as the availability view for each product.
