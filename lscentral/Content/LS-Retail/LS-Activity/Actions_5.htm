Actions - Reservation Card

This table lists and describes the actions on the Rerservation Card:

Section Action Description
Manage Search & New Client Use this action to search for the contact in the Member Contact list, and, if not found, create a new one and assign on the reservation.
Edit Open Activity Card Opens the Activity Card for the selected activity line.
  Schedule Opens the Reservation Schedule where you can assign resources and specific dates and times related to the reservation but not on a specific activity. Note: Commonly used for events or course schedule.
  Reservation Attributes To edit the attributes associated with the reservation.
  Activity Attributes To edit the attributes associated with the selected activity line.
  Tasks To edit the tasks associated with the current reservation and assign them to a specific employee.
  Transfer Charges Transfer unpaid additional POS charges to another reservation.
Status Confirm To assign Confirm status. If more than one Confirm statuses are possible, the user will be prompted with the list of options.
  Cancel To assign Cancel status. If more than one Cancel statuses are possible, the user will be prompted with the list of options.
  Change Status View all status options and assign the selected status.
View Matrix View the Resource Matrix availability based on the reservation date. Note: Any activities created while in the matrix view will be assigned to the current reservation.
  Resource Status View the Resource Status page.
  External View information related to the reservation based on external setup. Used for partner customization possibilities.
  Actual Consumption View the item inventory cost of all additional retail items consumed through the additional charge and consumption.
CRM Contact Card Opens the client's contact card with related information.
  Send Emails Select an email template and send to clients or resources according to the template settings.
  Quote Converts the reservation to a Business Central quote. See Reservation Types, Customer Account, the setup field that controls if this functionality is shown.

The system will create a standard Business Central quote based on the activity lines and additional charges. Every time this option is selected, the quote is updated with the latest reservation settings. Note: The quote is not deleted or posted at any time, so the user must attend to that manually.
  Invoice Converts the reservation to a Business Central invoice. See Reservation Types, Invoicing, the setup field that controls if this functionality is shown.

The system will create a standard Business Central invoice based on the activity lines and the additional charges. Only lines which are not assigned as FULLY PAID are inserted to the Invoice lines.

The invoice is updated every time this function is selected, according to the latest reservation settings. When the invoice is posted through normal Business Central posting features, the system requires that the reservation has been confirmed at that stage.

The Business Central posting process will update the reservation payment status per line, and maintain the link between the reservation and the posted invoice.
  Unit Tracking Opens the unit tracking worksheet to assign or return rental units. Note: If opened from the Reservation Card's action menu, the user can only return units, since assigning units can only be done from a specific activity.
