Membership Setup

This section describes the setup that must be in place before you can access any functionality in the LS Membership module.

General FastTab

Web Settings FastTab

You can access the setup from the Role Center.

The table below describes the main fields on the Membership Setup page.

General FastTab

Field Description
New Customer Template The customer settings to be used as defaults when the Membership module creates a new customer account.
New Customer Creation The available options here are Manual or At Subscription Batch Creation. Specifies if the customer account associated with each member should be created manually or automatically when the membership invoice is created.

The customer will get the same number assigned as the Member Contact No.
Default Member Club The member loyalty club to be assigned by default to new members.
Location (Invoice) The default location to assign on invoices.
Role  Cnt-X No. of Days Filter The Membership Role Center shows cues for new memberships assigned, cancelled memberships, and expiring members. This setting defines the date range filter for these indicators.
Access Control Export The codeunit which should be used for integrating with external gate or access control hardware. For reference see sample codeunit 10015903.
Subscription Price Group If memberships are assigned retail pricing, based on the related retail item, this value specifies which price group to use. Can be left blank.
New Posting Item (Template) The retail item to be used for default settings when assigning retail items to membership products (types).
Customer Account Link Defines how the customer accounts should be linked to the member contact. Either use the same member contact number for the customer account number, or use the standard Business Central business relations settings.
Final Subscription Charging Available options: Charge Full Period or Charge Until Expired Only.

When the subscription process is executed and the membership expires during the period being processed, specify if the client should be charged for the full period or if the last charge should be calculated as a proportion of the total number of days in the period, based on the membership expiry date.

Web Settings FastTab

Field Description
Web Service On This setting controls whether the membership API responds to external requests.
Activate Subscription Status The status setting to be used when subscription memberships are issued and activated on the web front end.
Cancel Membership Status The status setting to be used when memberships are canceled on the web front end.
Activate Single Sale Status The status setting to be used when single-sale memberships are activated on the front end.
