Using Attribute Groups

You can group attributes together and work with them at the same time through the group. You can show values for all the attributes in a group on the POS. The order of attributes in the panel is defined by the attribute group.

An attribute can be assigned to multiple groups.

To create attribute groups

  1. Go to the Attribute Groups page.
  2. Click the New action to create a new attribute group.
  3. Fill in the fields on the General FastTab as follows:
    FieldDescription
    CodeThis field contains the code that identifies the attribute group. You can use up to 20 characters.
    DescriptionThe description of the attribute group. You can use up to 50 characters.
    Value TypeThis field indicates which value type the attributes assigned to the group must be using. The options are:
    • No type specified - The attributes can have any value type
    • Text
    • Numeric
    • Amount
    • Date
    • File
    • Option Value
    • Table Link
    • Weighted Option

To assign existing attributes to the attribute group

  1. On the Attributes FastTab on a new line, select the desired attribute in the Attribute Code field. If the attribute group has value type restrictions, only attributes with the group's value type appear in the look-up list.
  2. Repeat to assign more attributes to the group.
  3. Change the order of attributes if necessary by selecting a line and clicking Manage - Up or Down.

Note:  The order of the attributes in this list determines the order of the attributes shown in panels on the POS.

To create new attributes and assign them to a group

  1. Go the Attributes page.
    Tip: You can open the page from the Attribute Groups and Attribute Group Card pages by selecting clicking the List action.
  2. Click New to create a new attribute, or select an attribute to use as a base for the new attribute and click Process - Create New From.
  3. Set up the attribute as desired.
  4. On the Group Assignments FastTab, select the desired attribute group in the Attribute Group Code field.

To automatically link attributes in a group to records and/or assign default values

For all the attributes in a group with Automatic linking, you can link the attributes to records according to their attribute linking lines.

Note: Automatic linking is not possible for the Dim. Base Value link type.

To link/assign default values for all link types

  1. On the Attribute Group page, click Actions - Link/Assign Default Values.
  2. If the attributes have a default value and there are attribute already linked to records, confirm whether or not you want the program to override existing values with default values. This is useful if you selected a wrong default value for the attributes or you extended the linking to set default values by item category and product group after you ran this function.
  3. The next confirmation tells you exactly how many records will be affected by running the function. Confirm the question.
  4. Click OK for the final message.

To assign attribute values to records

It is easy to assign attribute values to multiple records from the Attribute Assignments page.

  • On the Attribute Group page, click Actions - Assignments.

In the Attribute Assignments page you can:

  • Edit the Attribute Value field as is allowed for each attribute.
  • Filter on the Attribute Code field (the system, however, removes the group filtering at the same time).
  • Filter on the grouping fields for item and variant link types.
  • Filter on the No. field.
  • Edit hard attributes in this page. The Item, Store, and Extended Variant Values tables will be updated accordingly.
  • View the page for the record that is linked to the entry by clicking the Navigate action, then the View Linked Record action.
