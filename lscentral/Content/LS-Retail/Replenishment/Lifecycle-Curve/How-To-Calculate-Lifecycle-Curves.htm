How to: Calculate Lifecycle Curves

A Lifecycle Curve can be calculated based on the historical sales data of an item or item groups. The calculation can consider the Turnover (sales amount) or sold Quantity from the sales history.

Lifecycle Curves can be defined in three different Period Types: Day, Week, and Month.

To calculate a Lifecycle Curve

  1. Click the icon, enter Lifecycle Curves, and select the relevant link.
  2. Select an existing Lifecycle Curve, and click Manage - Edit, or create a new Lifecycle Curve by clicking the New action.
  3. Fill in the Lifecycle Curve Code, if the Lifecycle Curve is new.
  4. Select a Period Type.
  5. Enter No. of Periods and the system will create the Lifecycle Curve Lines based on the defined number of periods.
  6. Go to the Calculation FastTab, and select the Calculation Type.
  7. Select a value in the Date Calculation Type field, either Date or Date Formula. Date Formula is more dynamic compared to Date, because you are not required to specify any Start Date and it will always calculate backwards from the current WORKDATE, based on the defined Date Formula for Start Date.
  8. Enter a Start Date that is applicable for Date Calculation Type Date and for Date Formula for Start Date, if Date Calculation Type Date Formula is used.
  9. Enter values in the filter fields to narrow the calculation down to a specific item and locations.
  10. Click the Calculate action.
Field Description
Period Type Specifies whether the Lifecycle Curve is calculated over a day, week, or month.
No. of Periods Specifies the number of periods over which the Lifecycle Curve spans.
Calculation Type Specifies whether the Lifecycle Curve calculation will based on Quantity or Turnover.
  • Quantity - Quantity sold of the selected historical item(s).
  • Turnover - Sales amount of the selected historical item(s).
Date Calculation Type Specifies whether the starting date for the Lifecycle Curve calculation will be input directly or calculated based on the defined date formula.
Start Date This field is only visible if Date Calculation Type Date is used. It specifies the starting date of the Lifecycle Curve calculation.
Date Formula for Start Date This field is only visible if Date Calculation Type Date Formula is used. It specifies the formula to calculate the starting date of the Lifecycle Curve calculation.

The starting date will always be calculated based on the current WORKDATE.
