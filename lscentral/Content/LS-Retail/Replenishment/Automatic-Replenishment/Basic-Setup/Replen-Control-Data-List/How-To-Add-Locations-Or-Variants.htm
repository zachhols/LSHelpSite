How to: Add Locations or Variants

The Add Locations/Variants action (click the Manage action) on the Replen. Item Store Recs FastTab in the Replen. Control Data List lets you easily set up and create Replen. Item Store Records by selecting the locations and item variants for the selected item in the Items FastTab.

To add Locations or Variants into Replen. Item Store Rec

  1. Click the icon, enter Replen. Control Data List, and select the relevant link.
  2. In the Replen. Control Data List, select an item from the Items FastTab.
  3. On the Replen. Item Store Recs FastTab, click Add Locations/Variants action to open Location and Variant Selection page.
  4. In the Location and Variant Selection page, fill in the Active From Date field in the General FastTab, if necessary. This field will then be populated into Replen. Item Store Recs.
  5. Select the desired location(s) in the Selected check box in the Location Selection FastTab. Tip: You can apply filters for easy selection of locations.
  6. Select the desired variant(s) in the Selected check box in the Variant Selection FastTab. Tip: You can apply filters for easy selection of variants.
    Note: The Variant Selection FastTab is not visible, if the selected item does not have any variant.
  7. Click the Add Item Store Records action to open the Replen. Item Store Rec. Preview page.
  8. The selected locations and item variants will then be populated into Replen. Item Store Rec. Preview page.
  9. In the Replen. Item Store Rec. Preview page, select the Overwrite Replen. Item Store Rec. check box if the existing Replen. Item Store Rec should be replaced by new record, and select the Update Item Distribution check box if the Item Distribution should be updated.
  10. Click the Create Item Store Records action to populate the selected records in the preview list into the Replen. Item Store Recs FastTab in the Replen. Control Data List page.

To toggle the view of Locations or Variants List

  • Click the Location Selection drop-down arrow: 

    Select/Deselect All
    This action selects or clears the Selected check box for all the locations or variants that are within the filter.

    Show Selected
    This action shows all locations or variants that are selected in a Selected check box. At the same time, all the filters will also be reset.

    Show All
    This action shows all locations or variants and resets all filters.

Below is a description of the available options on the Location and Variant Selection page.

General FastTab

Field Description
Active From Date The Active From Date in Replen. Item Store Recs.

Location Selection FastTab

Field Description
Active for Replenishment The filter for location with Active for Replenishment in the Location Card. Available options are:
  • <BLANK>- no filter applies.
  • Yes - show location(s) that have the Active for Replenishment check box in the Location Card selected.
  • No - show location(s) that have the Active for Replenishment check box in the Location Card cleared.
In Item Distribution The filter for a store's location(s) listed in the Item Card's Distribution FastTab. Available options are:
  • <BLANK>- no filter applies.
  • Yes - show location(s) listed in the Item Card's Distribution FastTab.
  • No - show location(s) not listed in the Item Card's Distribution FastTab.

Note: The item distribution must have the following setup:

  • Status = Active
  • Ordered By = Central
  • Ordering Method = Calculate
Store Group The filter for store group.

Variant Selection FastTab

Field Description
Variant Dimension 1 The filter for the Variant Dimension 1 value for the selected item in the Replen. Control Data List page.
Variant Dimension 2 The filter for the Variant Dimension 2 value for the selected item in the Replen. Control Data List page.
Variant Dimension 3 The filter for the Variant Dimension 3 value for the selected item in the Replen. Control Data List page.
Variant Dimension 4 The filter for the Variant Dimension 4 value for the selected item in the Replen. Control Data List page.
Variant Dimension 5 The filter for the Variant Dimension 5 value for the selected item in the Replen. Control Data List page.
Variant Dimension 6 The filter for the Variant Dimension 6 value for the selected item in the Replen. Control Data List page.

Note: The visibility and caption of the Variant Dimension 1 to 6 fields in the Variant Selection FastTab are defined in Extended Variant Dimensions (matched based on Item No. and Framework Code).
