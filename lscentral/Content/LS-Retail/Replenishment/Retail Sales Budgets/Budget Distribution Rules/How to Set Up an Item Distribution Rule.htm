How to: Set Up an Item Distribution Rule

An Item Distribution Rule lets you define the distribution of budget values between items which are grouped by Division, Item Category, Retail Product Group, or Item Hierarchy. By using the Item Distribution Rule, the total budget value in the Retail Sales Budget will be automatically distributed among the items according to the assigned Item Distribution code.

To set up an Item Distribution Rule

  1. Click the icon, enter Retail Budget Distribution List, and select the relevant link.
  2. Click New to create a new Retail Budget Distribution.
  3. Fill in the Code and Description fields.
  4. In the Distribution Type field, select Item. The default value is Date.
  5. Select the relevant distribution calculation in the Type field. The default value is Equal:
    • Equal - Distributes an amount equally between the items which grouped by Division, Item Category, Retail Product Group, or Item Hierarchy.
    • Proportional - Distributes an amount based on the distribution of the existing values among the items.
    • Curve - Distributes an amount based on manual weight assignment.
  6. If you have the Type field set to Curve, the fields under Item Distribution Settings group will be visible. Fill in the Budget Line Type field, the available options are: Division, Item Category, Product Group, or Item Hierarchy. The Distribution Rule Lines will be one level below the defined budget line type.
  7. Click the Create Lines (Items) action to create the Budget Distribution Lines based on the Item Distribution Settings. The created lines are one level below the defined Division, Item Category, Product Group, or Item Hierarchy with Weight defaulted to 1.
  8. Manually reassign the Weight as desired.

Note: You can use the Calculate Item Dist. Curve action in Retail Sales Budget Lines to help you to capture the existing value in the budget and fill in the Weight fields in the Curve based Item Distribution Lines. For more information see How to: Edit Sales Budget Entries.
