How to: Edit Sales Budget Entries

One of the methods to maintain a Retail Sales Budget is to enter the budgeted quantities and amounts manually for each period.

To edit a budget from the matrix

To edit a budget from the Edit Budget page

See also

If a Retail Sales Budget is to be calculated from historical sales data, see How to: Create a Sales Budget from Sales Data for more information.

To edit a budget from the matrix

  1. Click the icon, enter Retail Sales Budget List, and select the relevant link.
  2. Select a budget from the list, and click the Sales Budget action to open the Retail Sales Budget page.
  3. You can use the Analysis Option field to switch the view and edit the value between Sales Amount, Quantity, Cost Amount, Discount Amount, Negative Adjustment, Closing Stock Value, and Opening Stock Value.
  4. To edit an entry, you must ensure that the View field is set to Budget.
  5. In the Lines FastTab, click Manage - Expand/Collapse on an entry to expand, or click Expand All to expand all entries at once.
  6. Next, select the entry that you would like to edit.
  7. Enter the desired value into the cell of the matrix.
  8. You can edit the value for all level of nodes. To edit a non-leaf node or total value, you must ensure that an Item Distribution and Date Distribution is assigned. The Store Distribution is required when the Budget Per Store is set to True in Retail Sales Budget. To set up a Budget Distribution Rule, see Budget Distribution Rules.

Note:
For a distribution rule with Curve calculation type, you can use the actions Calculate Item Dist. Curve, Calculate Store Dist. Curve, or Calculate Date Dist. Curve to help you to calculate the sum of the value in the budget and fill in the Weight in the selected distribution rule defined in the matrix entry. This is very useful when you would like to use the sales trend of a selected period based on the Sales Start Date and No. of Period defined in the Retail Sales Budget as a reference in distribution rule.

Besides that, you can use the actions Copy Item Dist. to lower lines, Copy Store Dist. to lower lines, or Copy Date Dist. to lower lines in the Lines FastTab to help you to easily copy the distribution rules in the selected lines to the child nodes. The action Clear all Dist. settings can be used to reset all the distribution rules defined in the Retail Sales Budget.

For a budget with Budget Per Store set to True, you can use the Store No. field in the Retail Sales Budget page to filter the budget or create the retail sales budget entry for the selected store.

To edit a budget from the Edit Budget page

  1. Click the icon, enter Retail Sales Budget List, and select the relevant link.
  2. Select a budget from the list, and click the Sales Budget action to open the Retail Sales Budget page.
  3. In the Lines FastTab, click Manage - Expand/Collapse on an entry to expand, or click Expand All to expand all entries at once.
  4. Select the entry that you would like to edit.
  5. Click Create - Edit Budget to open the Edit Retail Sales Budget page.
  6. Select the desired Item Distribution, Store Distribution, and Date Distribution. If a non-leaf node is edited, the entered amount will be distributed to its child nodes based on the selected Store Distribution, Item Distribution, and Date Distribution. The Store Distribution will only be applicable, if the Budget Per Store of the selected Retail Sales Budget is set to True. To set up a Budget Distribution Rule, see Budget Distribution Rules.
  7. Fill in the Sales Amount, Quantity, Cost Amount, Discount Amount, Negative Adjust Amount, and Closing Stock Value fields.

Note:
Item Distribution is visible when you are editing a non-leaf node. The Store Distribution and Date Distribution will only be visible if you are editing a leaf node.

For a budget with Budget Per Store set to True, you can use the Store No. field in the Edit Retail Sales Budget page to filter the budget or create the retail sales budget entry for the selected store.
