How to: Set Up Store Capacity

Store Capacity is used to maintain maximum item quantity or space consumption of sections and shelves in stores, grouped by item categories or product groups as well as for a specific season.

The fundamental setups of Store Capacity include:

  1. Item Capacity Value on the Retail Item Card, which is the space consumption per unit of the item (Base Unit of Measure).
  2. Section on the Store Section page.
  3. Shelf on the Section Shelves page.
  4. Store Capacity Template, which facilitates batch processing of Store Capacity lines by grouping them together.
  5. Goal on the Store Capacity page, which is the store capacity limit for the items.

To set up the Item Capacity Value

  1. Click the icon, enter Retail Item List, and select the relevant link.
  2. Select an item in the list, and click Manage - Edit.
  3. On the Merchandising FastTab, fill in the Item Capacity Value field. This value indicates the space consumption per item (Base Unit of Measure) for any store.

    4. Note: If the Store Capacity is only set up for a specific season, you must ensure that the Season Code field on the Retail Item Card, (Merchandising FastTab) is filled for all affected seasonal items.

To set up the Section and Shelf

  1. Click the icon, enter Store List, and select the relevant link.
  2. Select a store in the list.
  3. Click Navigate - Store - Sections to open the Store Section page.
  4. Fill in the Code, Description, Section Size and Size In fields.
  5. In the same Store Section page, click the Shelves action to open the Section Shelves page.
  6. Fill in the Code, Description and Section % fields.

To set up the Store Capacity Template

  1. Click the icon, enter Store Capacity Template List, and select the relevant link.
  2. Click New to create a new Store Capacity Template. This opens the card page.
  3. Fill in the Code and Description, and specify the fields in the Fields Settings FastTab:

    4. Field Description
    Show Values As This field specifies whether the fields in the Store Capacity list will be shown as actual or effective values.
    Show New Capacity This check box specifies if the capacity fields calculated with the linked Replenishment Journal will be shown in the Store Capacity list.
    Show New Inventory This check box specifies if the inventory fields calculated with the linked Replenishment Journal will be shown in the Store Capacity list.
    Default Capacity % Warning (Lower) This field specifies the default threshold value for the lower limit. The Capacity % or New Capacity % fields will be highlighted if their values are lower than or equal to the threshold value.
    Default Capacity % Warning (Upper) This field specifies the default threshold value for the upper limit. The Capacity % or New Capacity % fields will be highlighted if their values are higher than or equal to the threshold value.

To set up Store Capacity

  1. Click the icon, enter Store Capacity Template List, and select the relevant link.
  2. Create a new template or choose an existing template from the list. For more details see To set up the Store Capacity Template.
  3. Click the Edit Store Capacity action or drill down the No. of Lines field to open the Store Capacity page for the selected Store Capacity Template.
  4. Click New to create a new Store Capacity line.
  5. Fill in the Location Code field.
  6. Fill in either the Section Code or Item Category Code field. For more details on how section and shelf are assigned to product groups or items see Sections and Shelves.
  7. Fill in the Shelf Code, Product Group Code, and Season fields, if needed.
  8. Fill in the Goal field for the capacity goal.
  9. Fill in the Replen. Template Code and Replen. Batch No. fields, if you want to link the Store Capacity line to a Redistribution Replenishment Journal. For more information see How to: Assign Stock Transfer Based on Store Capacity.
  10. The Capacity % Warning (Lower) and Capacity % Warning (Upper) can be changed as needed, otherwise the values are set according to the default capacity percentage warning values in the Store Capacity Template.
    When a Capacity percentage such as Capacity % or New Capacity % crosses the lower or upper thresholds, the field will be highlighted in a different color.
  11. Repeat steps 4 to 10 for each Store Capacity you want to create.
  12. Click the Calculate Capacity action to calculate and update the values for all inventory, capacity, and capacity % related fields. The Last Time Calculated field indicates the date and time of the last Calculate Capacity action performed. For more details refer to How to: Calculate Store Capacity.

You can also access the Store Capacity page with the Store Capacity action, which is available on:

  • the Locations page.
  • the Retail Item Categories page.
  • the Retail Product Groups page.
  • the Seasons page.

    • Note: If you set up the Store Capacity from any of these pages, you must also specify the Store Capacity Template Code for all the Store Capacity lines you define.

To add Store Capacity by location

  1. Click the icon, enter Store Capacity Template List, and select the relevant link.
  2. Create a new template or choose an existing template from the list. For more details refer To set up the Store Capacity Template.
  3. Click the Edit Store Capacity action or drill down the No. of Lines field to open the Store Capacity page for the selected Store Capacity Template.
  4. Click the Add Lines by Location action to bulk create store capacity entries by location.
  5. Select the Store Capacity Template Code to which you want the store capacity entries to be added.
  6. Select the Include Sections check box, if you want to add store capacity entries by store sections.
  7. Select the Include Shelves check box, if you want to add store capacity entries by sections shelves.
  8. Select the Include Product Group check box, if you want to add store capacity entries by Product Group Section Location.
  9. Select the Overwrite Existing check box to overwrite the same Store Capacity entry.
  10. Click OK to run the report.

    11. Note: At least one of the Include options, the Include Sections, Include Shelves, or Include Product Group check box must be selected.
