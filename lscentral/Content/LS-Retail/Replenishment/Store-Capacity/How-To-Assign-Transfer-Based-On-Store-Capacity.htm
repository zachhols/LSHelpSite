How to: Assign Stock Transfer Based on Store Capacity

Store Capacity is used to maintain maximum item quantity or space consumption of sections and shelves in stores, grouped by item categories or product groups as well as for a specific season.

You can use the Store Capacity management tool to easily identify stores with underutilized spaces, and if needed, initiate stock transfer from the warehouse to fill up the stores. It is possible to link a Store Capacity line to the Redistribution Replenishment Journal so that you can use it to easily assign stock transfer from the warehouse, and finally, create Transfer Orders to replenish the stores.

To link the Store Capacity to Redistribution Replenishment Journal

  1. Click the icon, enter Store Capacity Template List, and select the relevant link.
  2. Create a new template or choose an existing template from the list.
  3. Click the Edit Store Capacity action or drill down the No. of Lines field to open the Store Capacity lines defined under the selected template.
  4. Select an existing Store Capacity line or create a new one. For more information, please refer to How to: Set Up Store Capacity.
  5. Fill in the Replen. Template Code field. You can only assign the Store Capacity line to a Replen. Template with Replenishment Type set to Redistribution, and Redistribution Calculation Type set to Manual Redistribution.
  6. Fill in the Replen. Batch No. field. You must assign a unique Replen. Batch No. to each of the Store Capacity lines under the same Replen. Template Code.
  7. Alternatively, you can click the Create Child Batches action under the Replenishment action group to automatically create the replenishment batches and assign them to the Store Capacity lines.

    8. Note: To use the Create Child Batches function, a master batch must be set up first, which will act as a reference to copy the settings for the subsequent batches which are created automatically through the function.

To set up the Master Replen. Batch

  1. Click the icon, enter Replen. Template List, and select the relevant link.
  2. Select the Replen. Template which you want to set up for your Store Capacity and open the card for editing, or create a new template. You can only assign a Store Capacity line to a Replen. Template with Replenishment Type set to Redistribution, and Redistribution Calculation Type set to Manual Redistribution.
  3. In the Replen. Template page, click Navigate - Template - Worksheet Batches. This opens and shows a list of batches currently assigned to the Replen. Template.
  4. Select an existing batch which you want to set up as the master and open it for editing, or create a new batch.
  5. In the Replen. Worksheet Batch page, activate the Master Batch check box under the General FastTab.

    6. Note: You can use the Copy Settings to Child action to copy the settings from the master batch to the child batches.

To assign Transfer Quantity based on Store Capacity

  1. Click the icon, enter Store Capacity Template List, and select the relevant link.
  2. Create a new template or choose an existing template from the list.
  3. Click the Edit Store Capacity action or drill down the No. of Lines field to open the Store Capacity lines defined under the selected template.
  4. Select an existing Store Capacity line or create a new one. For more information, please refer to How to: Set Up Store Capacity.
  5. Make sure the Replen. Template Code and Replen. Batch No. for the selected Store Capacity line are filled in.
  6. Click the Open Journal action under the Replenishment action group to open the respective journal.
  7. In the Redistribution Replenishment Journal, click the Add Items to Preview action under the Functions action group. This opens the Add Items to Redist. Journal report. This report will calculate the Redist. Preview according to the items and destination store location as set up in the selected Store Capacity line. The source location for the stock transfer will always be the warehouses. For more information on using this function, please refer to How to: Calculate Redist. Preview .
  8. Click the Show Redist. Preview action under the Functions action group. This opens the Redistribution Journal Preview page.
  9. Select the item that you want to initiate the stock transfer.
  10. Click the Show Items Availability action under the Process action menu. This opens the Retail Item Variant Availability by Location page, where you can assign stock transfer from the warehouse to the store location.
  11. Once you have assigned the transfer quantity for all the desired items, click the Add Items to Journal action under the Process action menu. This adds the items to the journal.
  12. The item inventory and capacity added in the Redistribution Replenishment Journal are shown in the Items in Journal (Inventory) and Items in Journal (Capacity) fields in the Store Capacity line.

    13. Note: You can always initiate the Store Capacity calculation by using the Calculate Capacity action under the Process action menu in the Store Capacity List. For more information, please refer to How to: Calculate Store Capacity.
