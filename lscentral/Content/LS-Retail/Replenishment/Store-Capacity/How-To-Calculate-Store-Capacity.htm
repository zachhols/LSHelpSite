How to: Calculate Store Capacity

The Calculate Store Capacity is a functionality that lets you calculate the space consumption of sections and shelves in stores.

Capacity Goal can be set up on the Store Capacity page per item category or product group, as well as for a specific season. See How to: Set Up Store Capacity for more information on setup.

The Calculate Store Capacity functionality will update all inventory and capacity fields on the Store Capacity page. Alternatively, running the Store Capacity report will also update these fields. The calculation can be performed manually in the Store Capacity page, or automatically with a Scheduler Job.

To calculate capacity on the Store Capacity Page

  1. Click the icon, enter Store Capacity, and select the relevant link.
  2. Click the Calculate Capacity action.

Below is the list of fields that are updated with the Calculate Capacity action. Whether the fields will be shown as actual or effective values is determined by the Show Values As field as defined in the Store Capacity Template.

Field Description
Inventory This field shows how many units, such as pieces, boxes, or cans, of the items are in inventory.
Eff. Inventory This field shows the inventory of the items, which may include the quantities from Sales, Purchase or Transfer Orders, depending on the setup in the Effective Inventory FastTab, in Replenishment Setup.
Items in Journal (Inventory) This field shows the inventory that will be added based on the quantity in the linked replenishment journal.
New Inventory This field shows the total of Inventory and Items in Journal (Inventory).
New Eff. Inventory This field shows the total of Eff. Inventory and Items in Journal (Inventory).
Actual Capacity This field shows the actual used capacity based on the Inventory of the items. This is calculated by multiplying the inventory of the items with their respective capacity value.
Actual Eff. Capacity This field shows the actual used capacity based on the Eff. Inventory of the items, which may include the quantities from Sales, Purchase or Transfer Orders, depending on the setup in the Effective Inventory FastTab, in Replenishment Setup.
Item in Journal (Capacity) This field shows the capacity that will be added based on the quantities in the linked replenishment journal.
New Capacity This field shows the total of Actual Capacity and Items in Journal (Capacity).
New Eff. Capacity This field shows the total of Actual Eff. Capacity and Items in Journal (Capacity).
Capacity % This field shows the ratio of the items' actual capacity to the defined store capacity limit.
Eff. Capacity % This field shows the ratio of the items' actual effective capacity to the defined store capacity limit.
New Capacity % This field shows the ratio of the items' new actual capacity to the defined store capacity limit.
New Eff. Capacity % This field shows the ratio of the items' new effective capacity to the defined store capacity limit.

To calculate store capacity with a Scheduler Job

  1. Click the icon, enter Scheduler Job List.
  2. Select and open the Scheduler Job with Job ID REPLEN-STORECAPACITY, which is already set up with the Calculate Used Capacity (10012322) report, the object responsible for performing the Store Capacity calculation. Alternatively, if you wish to create a new scheduler job, you can assign the report to the job with the Object Type and Object No. fields.
  3. Fill in the Scheduler Job Type Code field, the Time Units, Time Between Check, Next Check Date, and Next Check Time fields.
