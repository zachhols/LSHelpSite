One Warehouse

Replenishment can be run with one single warehouse. In such a case all stores will be supplied from that warehouse or directly from vendors, if set up accordingly.

Goal

The goals of this step:

  • Get a detailed understanding on the retailer's company structure and the warehouse structure
  • Create a topology drawing or a list of warehouses
  • Set up the warehouse in LS Central.

Output

  • Topology drawing, list of warehouses (see examples here):

    (Click to open)

LS Central setup considerations

Location

The relevant location has to be created and set up as a warehouse location.
The Location is a Warehouse check box on the Location card has to be selected:


(Click to enlarge)

For more details see Location.

Replenishment Setup

The warehouse location has to be set up as the default warehouse location in the Default Central Warehouse field on the Replenishment Setup page:


(Click to enlarge)

For more details see Warehouse.

Replenishment Templates

Purchase Replenishment Templates are used to calculate Purchase Order proposals for the warehouse.
The templates must have the value One Purchase Order per Vendor or One Purchase Order per Vendor with Cross Docking set up in the Purchase Order Type field:


(Click to enlarge)

For more information about Replenishment Templates see Replenishment Template.

Transfer Replenishment Templates are used to calculate Transfer Order proposals for the stores.
The templates must have the warehouse location code set up in the Location Code field:


(Click to enlarge)

For more information about Replenishment Templates see Replenishment Template.
