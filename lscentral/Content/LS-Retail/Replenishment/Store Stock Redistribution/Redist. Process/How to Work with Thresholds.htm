Store Stock Redistribution calculation is done in the following steps:

Thresholds are calculated after the creation of the Replen. Journal Lines and Replen. Journal Detail Lines. The calculation logic of Thresholds for Redistribution Replenishment Journals is similar to the calculation for Transfer Replenishment Journals. The Threshold calculation can be activated or deactivated in the Replen. Setup page.

To set up Threshold Rules for Redistribution

The setting of the Threshold Rules for Redistribution Replenishment Journals is similar to the Transfer Replenishment Journals. The only difference is the Type and the handling of Source No., as described below.

Type
A Threshold Rule for Redistribution Replenishment Journals is set up using the Type Redistribution, instead of Transfer, which is used for the Transfer Replenishment Journals.

Source No.
Similar to Transfer, a RedistributionThreshold Rule only accepts Location as a valid Source Type.
However, unlike Transfer, the Source No. for Redistribution Threshold Rule is not limited to warehouses only, meaning that it allows any location to be selected as the Source No.. This is because, using a Redistribution Replenishment Journal, the stock transfer could be from warehouse to store, or from store to store, depending on the Redist. Strategies and the setup for Warehouse in Redist. Matrix. For a Transfer Replenishment Journal, that is not the case, as it only accepts a stock transfer from warehouse to store.

Refer to How to: Set Up Thresholds for more information.

To activate Threshold Management

  1. Click the icon, enter Replen. Setup, and select the relevant link.
  2. On the Replen. Setup page, navigate to the Exception Management FastTab, select the Thresholds for Autom. Replen. check box.
