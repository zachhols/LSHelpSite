Allocation Plan Distribute lets you create the Allocation Plan Distribute Lines manually or with the system's assistance.

To add items to the Allocation Plan Distribute Lines manually

  1. Click the icon, enter Allocation Plan List, and select the relevant link.
  2. Open an existing Allocation Plan Distribute or create a new plan. For more information refer to How to: Create an Allocation Plan Distribute.
  3. In the Allocation Plan Distribute Lines FastTab, select an Item No. Then the Item Description, Allocation Rule Code, No. of Groups, No. of Destination, and Buffer % fields will be populated automatically.
  4. Select a valid Variant Dimension 1 Code if this is an item with variants.
  5. Enter the Qty. to Distribute.
  6. Change the quantities as needed. For more information refer to How to: Adjust Quantities in the Allocation Plan Distribute.

Fields in the Allocation Plan Distribute Lines FastTab:

Field Description
Item No. This field specifies the item number.
Variant Dimension 1 Code This field specifies the Variant Dimension 1 Code and is only filled for items with a Variant Framework.
If the lines were calculated by the Add Item Lines action, a separate line would be created for every Variant Dimension 1 Code.
Item Description This field shows the description of the item.
Allocation Rule Code This field specifies the currently active Allocation Rule that is applied for this line.
The Allocation Rule Code is inherited from the Allocation Rule Code in the General FastTab.
If the item has an Allocation Rule set up in the Allocation Rule Links, this value will be considered.
You can change the Allocation Rule in the Allocation Plan Distribute Lines.
No. of Groups This field shows the number of Store Groups or Customer Groups which are used to distribute the quantity.
This is accessed from the selected Allocation Rule.
No. of Destinations This field shows the number of Locations or Customers which are used to distribute the quantity.
This is accessed from the selected Allocation Rule.
No. of Dimensions This field shows the number of Dimensions which are used to distribute the quantity. This field is only used for items with a Variant Framework. This is accessed from the item's Variant Framework.
Qty. to Distribute This field shows the quantity of items which is assigned to be purchased and distributed to the destinations.
Qty. Distributed This field shows the result of the distribution calculation, after the Allocation Rule was applied to the Qty. to Distribute.
Usually, Qty. to Distribute and Qty. Distributed are equal.
Differences can occur if Maximum Quantity limits a full distribution of the Qty. to Distribute or if quantity rounding creates differences.
For more information refer to How to: Adjust Quantities in the Allocation Plan Distribute.
Buffer % This field specifies the percentage of the warehouse buffer for the affected item(s) that should be ordered additionally to the group quantities and kept in the warehouse. It is filled with the Warehouse Buffer % value on the General FastTab by default.
The value of this field can be changed in the lines and it affects the Buffer Qty. field.
Buffer Qty. This field specifies the buffer stock you would like to purchase additionally and keep in the selected warehouse.
This is how it is calculated:
Buffer Qty. = Qty. to Distribute * (Buffer % / 100)
Total Quantity This field shows the sum of the Qty. Distributed and Buffer Qty.
Minimum Quantity This field specifies the lower limit of the quantity for the destination lines.
Maximum Quantity This field specifies the upper limit of the quantity for the destination lines.
Group 1 Quantity This field shows the quantity distributed to the first group.
Group 2 Quantity This field shows the quantity distributed to the second group.
Group <n> Quantity This field shows the quantity distributed to the <n>-th group.
Create Documents This check box specifies if the Allocation Plan Distribute Lines should be considered in the Create Purchase Orders, Transfer Orders, Sales Orders action.
Whenever the documents are created successfully, the value of this field will be set to FALSE.
No. of PO Lines Created This field shows the number of lines created in Purchase Orders.
No. of TO Lines Created This field shows the number of lines created in Transfer Orders.
No. of SO Lines Created This field shows the number of lines created in Sales Orders.

Note: The Group Quantity fields allow a quick access to the quantities assigned to the Destination Groups.
Note: Only the first six groups are displayed as individual columns. To access more groups you can drill down in the No. of Groups field or make use of the Group Lines option in the Line menu, select the right group, and drill down the Qty. field.

To add items to the Allocation Plan Distribute Lines with the Add Item Lines function

  1. Click the icon, enter Allocation Plan List.
  2. Open an existing Allocation Plan Distribute or create a new plan. For more information refer to How to: Create an Allocation Plan Distribute.
  3. In the Allocation Plan Distribute page, click the Add Item Lines action.
  4. A request page is displayed, and you can define the items that need to be included in Allocation Plan Distribute Lines by using the available items' filters.
  5. Click OK. The system adds the items that meet the selected conditions to the worksheet.
  6. Enter Qty. to Distribute.
  7. Change the Qty. Distributed as needed. For more information refer to How to: Adjust Quantities in the Allocation Plan Distribute.

Note: The Add lines where Quantity on Hand is 0 field is only applicable to Buyer's Push.

Example 1:

  • You want to distribute 400 units of Appointment Diary by using Allocation Rule Code DEFAULT.
  • There are 4 Destinations in 4 Groups without any dimensions.
  • The Buffer % is 50.
  • A total of 600 units of Appointment Diary will be purchased but only 400 of them will be distributed to the destination locations as 200 units of them are buffer stock.
  • The Allocation Rule assigns 80 units to S0001, S0002, S0003 each and 160 units to S0004 as it has a higher weight defined in the Allocation Rule.

Example 2:

  • You want to distribute 400 units of Appointment Diary by using Allocation Rule Code DEFAULT.
  • There are 4 Destinations in 4 Groups without any dimensions.
  • The Buffer % is 50.
  • Minimum Quantity and Maximum Quantity are defined as 90 and 100 respectively.
  • Compared to the result in Example 1, the Allocation Rule increases the quantity assigned to S0001, S0002, S0003 to 90 each due to the Minimum Quantity and decreases the quantity assigned to S0004 to 100 due to the Maximum Quantity.
  • As a result, only 370 units are distributed and a total of 570 units of Appointment Diary will be purchased including the 200 buffer stock.

To see the distribution result on Group Line level

  • Navigate to the Allocation Plan Distribute Lines FastTab, and click Line - Group Lines to open the Allocation Plan Groups page.

The Allocation Plan Groups page shows the Destination Groups which are applied to the Allocation Plan Distribute Lines according to the selected Allocation Rule. You can change the Group Weight, if you want to use a different distribution share. The field % Share is calculated and represents the share of the individual Destination Group line of the total - the sum of all Destination Group lines should be 100%.

The Qty. field shows the total distributed quantity for the Destination Group and it is a drill-down field which opens the Allocation Plan Location Lines page. The Location Lines represent the most detailed level in the distribution.

The Sort Order can be used to reorder the sequence of the Group Quantity columns shown in Allocation Plan Distribute page.

To see the distribution result on Destination Line level

  • Navigate to Allocation Plan Distribute Lines, and click Line - Destination Lines to open the Allocation Plan Des. Lines page.

The Allocation Plan Des. Lines page shows the Destinations which are applied to the Allocation Plan Distribute Lines according to the selected Allocation Rule. You can change the Destination Weight, if you want to use a different distribution share. The field % Share is calculated and represents the share of the individual destination line of the total - the sum of all Destination lines should be 100%. This page also shows the field Group % Share which can be changed in the Group Line page.

The field Qty. shows the total distributed quantity for the Destinations and it is a drill-down field which opens the Allocation Plan Location Lines page. The Location Lines represent the most detailed level in the distribution.

Note: In the Allocation Rule Calc. Method field you can apply a different calculation method for the Destination lines. The calculation method will recalculate the Destination Weight and update the % Share.

To see the distribution result on Variant Line level

  • Navigate to Allocation Plan Distribute Lines, and click Line - Variant Lines to open the Allocation Plan Dim Values page.

The Allocation Plan Dim Values page shows the variants that are applied to the Allocation Plan Distribute Lines. You can change the Qty. to Distribute field, if you want to use a different distribution quantity.

The Distributed Quantity shows the total distributed quantity for the variant and it is a drill-down field that opens the Allocation Plan Location Lines page for the selected variant. The Location Lines represent the most detailed level in the distribution.

You can revise the value of Distributed Quantity by changing the Weight and clicking Distribute Quantity - Function in the Navigate action menu.

To see the distribution result on Location Line level

  • Navigate to Allocation Plan Distribute Lines, and click Line - Location Lines to open the Allocation Plan Location Lines page.

The Allocation Plan Location Lines page shows the locations (or customers) which are applied to the Allocation Plan Distribute Lines according to the selected Allocation Rule. The Location Lines represent the most detailed level in the distribution calculation and can be seen as the result on the lowest level. By default, the Variant Dimension 1 Code and Variant Dimension 2 Code are shown, but all six Dimension Codes are stored in the Location Lines.

You can change the Qty. field, if you want to use a different distribution share. If you change the quantity, the system will place a check mark in the Changed column. To preserve the changes from top-down recalculations, you must select the Locked check box.

You can change the Buffer Qty. field, if you want to use a different ratio between distributed quantity and quantity kept in the warehouse.

To add or delete an Allocation Rule group

You may add or delete an allocation group without making any changes in the Allocation Rule setup.

  1. Navigate to Allocation Plan Distribute Lines, and click Line - Group Lines to open the Allocation Plan Groups page.
  2. Insert a Group Code, Group Weight, and Sort Order.
  3. Close the page, repeat step 1 and select Destination Lines, this time to open the Allocation Plan Des. Lines page.
  4. Create a new line by selecting the Group Code created in step 2. Next, define the Destination Type, Destination Code, and Destination Weight.
  5. Close the Allocation Plan Des. Lines page.
  6. Revise or re-enter the value of Qty. to Distribute and the new quantity will be distributed accordingly.

Note: Adding or deleting an allocation group will result in removing the Allocation Rule Code in the Allocation Plan Distribute Lines.
