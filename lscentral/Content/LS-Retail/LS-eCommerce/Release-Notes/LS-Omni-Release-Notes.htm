LS Omni Suite (4.0) Release Notes

About This Version

The LS Omni solutions are integrated with LS Central (13.06), providing simplified apps with real-time connectivity through web services. The LS Omni solutions are:

  • Omni Service – version 4.0
  • Mobile POS (Android) – version 4.0
  • Mobile Inventory (Android) – version 4.0
  • App Shell – version 1.7

Note:  All released clients connect by default to a central LS Omni server hosted in the cloud. This server is connected to LS Central (13.xx) that is populated with demo data and acts as a source of data suitable for demonstrations and software evaluation. If you request a local installation of the LS Omni server for any of the solutions that you wish to host privately and connect to your LS Nav/LS Central solution, please contact LS Retail through Partner Support on the LS Retail Portal (log in required).

  • The LS Omni Server is Unit Tested against 2018 (11.xx) and LS Central (13.xx).
  • The Apps are only tested against LS Central (13.xx).
  • To use LS Omni against LS Nav 2016 (9.xx) and LS Nav (10.xx), use LS Omni version 3.xx.
  • To use LS Omni against LS Nav 2015 (8.xx), use LS Omni version 2.3.3. 
  • LS Nav older than version 8.x is not supported.

To be able to run older version of the apps, downloaded from mobiledemo.lsretail.com, against our demo server, the URL must be set to:

  • Version 2.4.x and up:

    http://mobiledemo.lsretail.com/LSOmniService/appjson.svc

  • Version 2.3.x and older:

    http://mobiledemo.lsretail.com/LSOmniServiceOld/json.svc

The Release Notes specify the new features and changes implemented in the LS Omni solutions in this release. For more information about the functionality, consult the LS Central Help for each solution. This LS Omni suite is part of LS Central (13.06). See the LS Central (13.xx) Release Notes for more details.

Documents included in the release are also available on the LS Retail Portal.

Minimum System Requirements

  • Requirements for LS Omni Host running on IIS:
  • 80 GB disk space, depends on LS Nav/Central database size
  • 4 GB Ram
  • Windows 7/8/10 Professional/Enterprise
  • Windows server 201x
  • IIS 7 and above
  • SQL Server 201x
  • SQL Server Advanced Express 201x

Note: The Full Text Search feature in the SQL Server is optional but strongly recommended.

Features and Fixes in This Release

Mobile POS (Android)

  • Added in LSKey support for omni Server
  • Small fixes for Adyen
  • Added in Feature Flags from LS Central
  • Auto refresh changed
  • Updated LS Pay version.

Inventory (Android)

  • Added in LSKey support for omni Server
  • Auto refresh changed.

Omni Server

  • Multi tenant support
    • Omni admin portal to manage tenants
  • Unified order/transaction lookup
  • Bug fixes.

LS Central App (Android)

  • Minor bug fixes
  • More logging added
  • Added support of Nav Username/Password Authentication
  • Updated LS Pay version.

LS Central App (iOS)

  • Minor bug fixes.
