To open the ScanPayGo profile

  • Click the icon, enter ScanPayGo Profile, and select the relevant link.

Information in this profile is used to set up and control the ScanPayGo Mobile app: to select a setup for Stores and Store Groups, and to select which Tender Types are valid for each profile.

The table below lists the fields in the profile.

Field Description
Code Unique code to identify each new ScanPayGo profile.
Description Short text to describe the usage of the profile.
Security Check Trigger Here you specify when to activate security check on the ScanPayGo app.

Available options are:

  • When selecting payment - Security check will be activated when the user selects Payment in the ScanPayGo app.
  • After Payment - Security check will be activated after a payment has been finalized on the ScanPayGo app.
  • Never - Security check will not be activated on the ScanPayGo app.

Profile Links

In the ScanPayGo Profile page, click the Profile Links action to work with the following fields:

Field Description
Link Type Set the type of link to be selected from in the Link Id.

Available options:

  • Store Group - Select the Store Groups to use the ScanPayGo profile.
  • Store - Select the stores to use the ScanPayGo profile.
Link Id Select the Store Groups, Stores, and Device types using this profile.
Device Type Select if the device running the ScanPayGo app is owned and provided by the Store, or if it is a customer device.

Tender Types

In the ScanPayGo Profile page, click the Tender Types action to work with the following fields:

Field Description
Tender Type Select valid Tender Types for the ScanPayGo app using this profile.
Description Description of the selected Tender Type.

Feature Flags

In the ScanPayGo Profile page, click the Feature Flags action to work with the following fields:

Field Description
Feature ID Create a new Feature Flag by typing in an ID. These Feature Flags are setup value pairs for the ScanPayGo app.
Feature Value A value for the setup pair Feature ID and Feature Flag.
Description A text describing the purpose of this Feature flag value pair.
