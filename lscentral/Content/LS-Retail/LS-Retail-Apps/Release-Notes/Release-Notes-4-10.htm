LS Retail Apps 4.10 Release Notes

About This Release

The LS Omni solutions are integrated with LS Central (14.02, 15.03 and 16.xx), providing simplified apps with real-time connectivity through web services. The LS Omni solutions are:

  • Omni Service – version 4.10
  • Mobile POS (Android) – version 4.10
  • Mobile Inventory (Android) – version 4.10
  • App Shell – version 1.17

Note: All released clients connect by default to a central LS Omni server that is hosted in the cloud. This server is connected to LS Central (16.xx), which is populated with demo data and acts as a source of data suitable for demonstrations and software evaluation.
To request a local installation of the LS Omni server for any of the solutions that you wish to privately host and connect to your LS Nav or LS Central solution, please contact LS Retail through Partner Support on the LS Retail Partner Portal (log in required).

LS Omni Server is unit tested against LS Nav 2018 (11.xx) and LS Central (13.xx , 14.xx, 15.xx, 16.xx). 

The apps are only tested against LS Central (16.xx).

 To use LS Omni against LS Nav 2016 (9.xx) and LS Nav (10.xx), use LS Omni version 3.xx.

To use LS Omni against LS Nav 2015 (8.xx), use LS Omni version 2.3.3. 
LS Nav older than version 8.x is not supported.

To run an older version of the apps, downloaded from mobiledemo.lsretail.com against our demo server, the URL must be set to:

  • Version 2.4.x and up:
    http://mobiledemo.lsretail.com/LSOmniService/appjson.svc
  • Version 2.3.x and older:
    http://mobiledemo.lsretail.com/LSOmniServiceOld/json.svc

These release notes cover new features and changes implemented in the LS Omni solutions in this release. For more detailed information about the functionality, consult the LS Central Help for each solution.
This LS Omni suite is a part of LS Central (16.0), see the LS Central (16.0) Release Notes for more information.

Minimum System Requirements

Requirements for LS Omni Host running on IIS:

  • 80 GB disk space, depending on the LS Nav/LS Central database size
  • 4 GB Ram
  • Windows 7/8/10 Professional/Enterprise
  • Windows server 201x
  • IIS 7 and above
  • SQL Server 201x
  • SQL Server Advanced Express 201x

Note: The Full Text Search feature in SQL Server is optional but strongly recommended.

The LS Central version mentioned in the list is the highest version within a major version that an Omni Version supports.  Each LS Central minor version, lower than the one mentioned in the list, is also supported. For example: Central (13.05) means that 13.04, 13.03, 13.02, 13.01, and 13.0 are supported, too.
