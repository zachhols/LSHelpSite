Printer Setup for Mobile POS

Note: Before you start the setup process, you must make sure that the computer connected to the OPOS printer has the Printing Station Monitor service installed and running.

To set up OPOS external printers on the Mobile POS app

To set up OPOS external printers for Mobile POS in LS Central

The Printing Station Monitor (which is an external print service) is used by the Mobile POS to reserve and claim printers. As the LS Central stationary POS does not use this external print service, the same printer cannot be used for both Mobile and stationary POS at the same time.

Setting up an OPOS (external) printer for the Mobile POS in LS Central is a process in three steps:

  1. First, you must create a new Hardware Profile for the OPOS printer the Mobile POS is going to print to. The profile should be marked as Printer Remote and the Printer Server Host should have the IP address of the machine running the Printing Station Monitor and the OPOS Printer.

  2. Then, you create a new POS External Printer (click the icon, enter POS External Printers, and select the relevant link.) Here you link the Hardware Profile yo created in Step 1.

  3. Finally, you open the POS Terminal Card for the Mobile POS, and on the Printing FastTab, you add the newly created POS External Printer ID.

These steps are described in detail below.

  4. Now everything should be connected and you can print from the mobile device.
