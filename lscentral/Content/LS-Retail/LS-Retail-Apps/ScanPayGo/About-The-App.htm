ScanPayGo branding for retailers

With version 2 of the ScanPayGo app, scheduled to be released in September/October 2021, LS Retail’s partners will be able to download the code for the ScanPayGo app and modify it so it will be branded  and adjusted to the needs and requirements of individual retailers.

Changes you can make in the app:

  • You can change the look of the app to have the retailer’s logo and colors.
    • There are two main colors in the app that you can easily change, the background color and another main color.
    • You can import the retailer’s logo into the app.
  • You can edit the payment options that the retailer wants to have in the app.
    • It is the retailer’s decision, if shoppers should be able to finalize their shopping in the app or not.
    • In the case where retailers do not allow shoppers to pay with a card or loyalty points in the app, partners can adjust the app for that situation.
    • You can specify which tender types can be used in the app.

Setup and version information

The ScanPayGo app works with LS Central and connects with LS Central using the LS Commerce Service.

  • Requires LS Central, version 18.0 or later.
  • LS Commerce Service, version 4.19 or later, must be installed and connected with the LS Central service.

Installation on mobile devices

The ScanPayGo app must be installed into each individual mobile device. The app is available for iOS and Android mobile operating systems.

When retailers have had the ScanPayGo app branded and changed to their liking, the app can be made available in Playstore for Android and in Appstore for iOS.

Demo version

You can install a demo version of the ScanPayGo app from the LS Retail App Center.

The demo version of the ScanPayGo app is by default connected with a demo setup of LS Central and is therefore available for everyone to test and demo. It is easy to change the settings in the ScanPayGo app to connect the app with a different setup. This is done in the app’s Settings.

The LS Central Demo Data company includes an example with a grocery store to be used with ScanPayGo. This is the demo store Store S0001 or Cronus Super Market South. Here you can use the Item view action to view the grocery items in the store.

Note: ScanPayGo can be used in different types of retail. The demo example is for grocery, but there is no restriction in the system for which type of retail the system can be used.
