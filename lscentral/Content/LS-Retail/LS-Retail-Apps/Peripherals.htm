EFT Support

The Mobile POS integrates with LS Pay which opens up support for all the payment providers that LS Pay has integrated with. Today these payment providers are:

  • Pay Ex
  • Verifone Point
  • Valitor (Android only)
  • Mock (for demo purposes)

This includes support for barcode scanners when available on the PED devices.

These settings can be configured in the Devices tab in the Settings screen on the Mobile POS.

Printer Support

Supported printers:

  • Zebra LinkOS
  • Epson Epos Bluetooth
  • Epson Epos Network
  • Bixolon SPP series
  • Zebra ZQ110

Note: These instructions apply to all LS Retail apps that can connect to printers. LS Retail recommends using EPSON Epos or Zebra LinkOS.

To connect printers to mobile apps

  1. Open your Android phone and navigate to Bluetooth settings.
  2. Search for a new device, looking for your printer, and then pair the printer and the phone together.
  3. When pairing the devices is done, open your app and navigate to the Settings screen.
  4. Open the Devices tab. Tap the drop-down arrow under Select printer to see a list of all supported printers.

  5. Select a printer to connect.

The sections below describe how to set up the different printers.

Zebra LinkOS

  1. When you have connected a Zebra LinkOS printer to your phone, open your app and select Zebra LinkOS from the drop-down menu.
  1. Select the printer name from the Select Bluetooth printer drop-down menu.

  2. You are now all set and ready to use the printer.

Zebra LinkOS custom fonts

Custom font support for Zebra LinkOS printers was added in Android AppShell 2.3 and LS Central 17.0.

To use custom fonts:

  1. Install the font on the printer (must be in .TTF format). For more information click here.
  2. Go to the settings for the printer in LS Central, in the POS Printer Card, and change the printer formats in the Printer Normal and Printer Bold fields on the Layout and Format FastTab. The format is [Font];[Height];[Width], for example [E:SOU001.TTF];[15];[12].

Epson Epos (Bluetooth)

  1. When you have connected an Epson Epos Bluetooth printer to your phone, open your app and select Epson Epos (Bluetooth) from the drop-down menu.
  2. Select the printer model from the Select printer model drop-down menu that lists all supported models.
  3. You can also select the printer language from the Select printer language drop-down menu.

  4. You are now ready to start using the app and the selected printer.

Epson Epos (Network)

  1. When you are printing on a network printer, you can open your app directly and navigate to the Settings screen.
  2. Open the Devices tab, and select a printer, which in this case is Epson Epos (Network).
  3. Enter the printer's IP Address. Note: Make sure that your printer and the app are using the same IP Address.
  4. Select the printer model and, optionally, the printer language from the drop-down menus.

  5. You are now ready to start using the app and the selected printer.

LS Central printing station

If you select LS Central printing station in the printer list, you will be using the LS Hardware Station to print from the app.

Bixolon SPP series and Zebra ZQ110

  1. If you connected your phone with either a Bixolon or Zebra ZQ110 printer, you can open the app directly and navigate to the Settings screen.
  2. Open the Devices tab, and select a printer from the Select printer drop-down menu, in this case Bixolon SPP series or Zebra ZQ110.
  3. You are all set and ready to use the selected printer.
