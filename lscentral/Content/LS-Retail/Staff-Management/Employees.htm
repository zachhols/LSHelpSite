Employees

The final setup for the Staff Management is to create the employees to be able to start planning and start using the time registration. The following fields are required:

  • No. - This field is filled automatically by the number series.
  • First Name - Employees first name.
  • Last Name - Employees last name.
  • Work Arrangement - Set this to FULLTIME.
  • Work Location - Set this to LOC1 (Note: the employee can still be used in multiple locations).
  • Time Entry ID - The ID used for time registration (Time Entry password is optional).
  • Work Role - Set this to CHEF (Note the employee can still have multiple roles).

Optional fields:

  • Work Shift: Only if the employee should be automatically assigned on this shift during planning.
  • Shift Pattern: Only if the employee should be automatically assigned according to this pattern during planning (if used, then leave work shift blank and vice versa).

Create three employees where you have one employee on no shift/pattern, then one employee set to fixed shift DAYSHIFT, and the 3rd that’s on the shift pattern PATTERN1.

Have the first employee assigned as CHEF, and the other two as WAITER in Work Roles. Make sure the employees have Time Entry ID assigned (1,2 & 3 for example).

Now you are all set to start your first planning and get the hours from the time registration.
