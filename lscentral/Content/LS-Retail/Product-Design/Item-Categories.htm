Item Categories

To set up Item Categories

To set up Open Item Category Sale

There are two more levels of Product Grouping in LS Central:

  • Item Categories: The higher level. Each Item Category contains a number of Retail Product Groups.
  • Retail Product Groups: The lower level of grouping. Each Retail Product Group contains a number of retail items.

It is important to have a well-defined Item Category structure, before you start to set up Item Categories. Before setting up Item Categories you should define how detailed the Item Category structure should be in order to serve best the needs of your business. Then you create Item Categories and assign closely related Retail Product Groups to the same Item Category.

To ensure the correct handling of retail items, you need to set up Item Categories. You can use Item Categories to:

  • Group Retail Product Groups
  • Define default Posting Groups
  • Base POS cost calculation on Item Categories
  • Collect and view statistics on Item Category level
  • Set up an Open Item category sale

To set up Item Categories

  1. Search for Retail Item Categories.
  2. Click the New action to enter a new Item Category.
  3. Fill in the Code and Description fields.
  4. Fill in the other fields, as needed.

  5. Repeat steps 2 to 4 for each Item Category you want to set up.

Microsoft Dynamics 365 Business Central does not allow a sale on open item groups. Only sales on items are possible. Nevertheless you can set up open item category sale or open retail product group sale, meaning that when you have an item that does not have an item number registered, you can sell the item as a part of the Item Category or a Retail Product Group.

Note: When selling an open item category sale, statistics will not be retained on item level, but on Item Category level, also; the inventory will not be updated.

To set up Open Item Category Sale

  1. In the Retail Item Card, create a retail item.
  2. In the Description field, insert a description of the Open Item Category sale.
  3. In the Item Category Code field, select the relevant Item Category.
  4. On the POS FastTab, select the option Must Key in New Price in the Keying in Price field.

  5. On the Invoicing FastTab, select the required fields.

When performing an Open Item Category sale, you select the item created above as the Item Category to be sold.

You can make a similar setup for a Retail Product Group.
