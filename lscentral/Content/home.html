About the Help

This is the online help and release notes for LS Central 18.4. As a reference, help and release notes for LS Nav 8-11, LS Nav/LS Central 13, LS Central 14, LS Central 15, LS Central 16, and LS Central 17 are also available here.

The help is divided into sections, two separate sections for LS Retail and LS Hospitality products, one for Microsoft Dynamics 365 Business Central help, one for release notes, and finally two separate sections for the LS Central Implementation Guide and the LS Central Installation Guide. Click the tiles to access these sections.

The help files are included in the LS Central installation package, so you can set the help up locally, if required.
